Horizon Curvature

The Horizon Curvature algorithm computes various horizon-based curvature attributes.

Overview of the Horizon Curvature Algorithm

Curvature analysis is a set of related - but different - surface attributes that you can compute for a horizon. These attributes are sensitive to subtle morphology changes, and can be useful interpretation aids. Some interpreters refer to them as "second order derivatives."

First order derivatives (such as Dip, Azimuth, and Edge-detect) can return faulty data at extreme (very high or low) dip points. Azimuth, for example, is unstable at low dips, and Dip becomes saturated at high dips. When a surface is steeply dipping, all parts of the surface have high dips and subtle local features are not apparent on dip maps. The "second order" derivatives do not suffer from these types of problems. Thus, second order derivatives often reveal small-scale subtleties more objectively than first order derivatives.

When calculated from a surface, such as a seismic horizon, the curvature value depends on the direction in which the curvature is measured. Positive curvature is associated with concave-down features, and negative curvature is associated with concave-up features.

See Horizon Curvature Attributes for details about the attributes offered in GeoCraft.

Another important aspect of curvature analysis is aperture. Aperture is the window that determines how much surface data is used in the calculations. Aperture is often referred to as wavelength. Short-wavelength (small aperture) curvature displays can be contaminated by noise. Longer wavelength (larger aperture) curvature displays tend to be smoother. Different scales of features may be superimposed on an image, so calculating curvature with different apertures may be useful in your analysis.

Performing Horizon Curvature Analysis on Data in GeoCraft

  1. Load horizon (grid) data into GeoCraft.
  2. Select a Horizon.
  3. Assign an Attribute parameter.
  4. Type in an Aperture value.
  5. Accept the Name, or type in a new one.
  6. Optional Type text in the Comments field. You will see this text in the Properties view when you select the new horizon in the Repository.
  7. Click Run.

GeoCraft runs the calculation, and the resulting horizon populates in the Repository under the Grids folder. You may now use the GeoCraft viewing tools to view the results of the curvature analysis algorithm.
