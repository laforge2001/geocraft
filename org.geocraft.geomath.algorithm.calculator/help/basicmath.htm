Calculator Basic Math Functions

The GeoCraft Basic Math calculator performs functions (such as adding, exponentiation, or finding the average value) on horizons loaded into GeoCraft.

GeoCraft Basic Math Calculator: an Overview

The GeoCraft Basic Math Calculator performs the displayed calculation for each function on the data selected for the b and c variables, and then writes the results of the calculation out as a new output variable.

Note: The Basic Math, Boolean Algebra, Scientific, and Wavelet Math calculators now use the Free Form Horizon and Wavelet Calculators. Therefore, when you click the Open: Free Form 3D Equation Calculator (Horizon) button, the Free Form Calculator pops up with the equation already inserted in the Editor. This allows you to modify the equation before pressing Run in the Free Form calculator.

Best Practices for Mathematical Operations in GeoCraft

When you perform a calculator function on two horizons, the two geometries must be identical. This means that each grid must contain an identical number of rows and columns, and the x and y locations of each node must be identical.

Using the GeoCraft Basic Math Calculator

The following procedure provides a basic overview of how to use the GeoCraft Basic Math Calculator. See each of the Basic Math Calculator related topics to learn how each function works.

Note: Operations are performed only on non-null values, unless otherwise noted in the Basic Math Calculator help topics.

  1. Load horizon (grid) data into GeoCraft.
  2. Select a b variable.
  3. Select a c variable (not necessary for the Absolute Value operation).

    4. Note: Variable c may be a number (for example, 2.5) instead of a horizon.

  4. Select The operation to perform:
    • b + c: Addition
    • b - c: Subtraction
    • b ** c: Exponentiation
    • b * c: Multiplication
    • b/c: Division
    • abs(b): Absolute Value
    • max(b,c): Maximum Value
    • min(b,c): Minimum Value
    • avg(b,c): Average Value
  5. Accept the default output name, or type a new name in the field.
  6. Click Run the current algorithm.

When you click the Run the current algorithm button, GeoCraft runs the calculation in the background. The answer to your calculation is put in the GeoCraft Repository .
