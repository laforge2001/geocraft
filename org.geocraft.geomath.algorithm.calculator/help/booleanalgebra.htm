Calculator Boolean Algebra Functions

The GeoCraft Boolean Algebra calculator applies logical or Boolean operations to horizons.

These Boolean operations can be used to merge horizons. However, only two horizons can be merged at a time. If you need to merge several horizons, consider using the Horizon Merge algorithm, which is part of the Horizon Utilities group.

GeoCraft Boolean Algebra Calculator: an Overview

The GeoCraft Boolean Algebra Calculator performs the selected operation on the data selected for the b and c variables, and then writes the results of the calculation out as a new output variable. The c variable may be a numeric constant. If the result of the operation is true, the result is set to the appropriate input variable. If the operation is false, the result can be set to Null.

Example: In this example, input b is top salt and input c is 1000. If the operation is b > c, then the result is true when top salt time exceeds 1000 and the top salt time is output. But when the top salt is less than or equal to 1000, the output will be Null.

Note: The Basic Math, Boolean Algebra, Scientific, and Wavelet Math calculators now use the Free Form Horizon and Wavelet Calculators. Therefore, when you click the Open: Free Form 3D Equation Calculator (Horizon) button, the Free Form Calculator pops up with the equation already inserted in the Editor. This allows you to modify the equation before pressing Run in the Free Form calculator.

Best Practices for Mathematical Operations in GeoCraft

When you perform a calculator function on two horizons, the two geometries must be identical. This means that each grid must contain an identical number of rows and columns, and the x and y locations of each node must be identical.

Using the GeoCraft Boolean Algebra Calculator

The following procedure provides a basic overview of how to use the GeoCraft Boolean Algebra Calculator. See each of the Boolean Algebra Calculator related topics to learn how each function works.

Note: Operations are performed on non-null values, unless otherwise noted in the Boolean Algebra Calculator help topics.

  1. Load horizon (grid) data into GeoCraft.
  2. Select a b variable.
  3. Select a c variable.

    4. Note: Variable c may be a number (for example, 2500) instead of a horizon.

  4. Select The Operation to Perform:
    • b == c: Equal Values
    • b != c: Unequal Values
    • b and c: Intersecting Values
    • b > c: Greater Than
    • b >= c: Greater Than or Equal to
    • b or c: Or
    • b < c: Less Than
    • b <= c: Less Than or Equal to
    • xor(b,c): Excluded Elements
  5. Accept the default output name, or type a new name in the field.
  6. Click Run the current algorithm.

When you click the Run the current algorithm button, GeoCraft runs the calculation in the background. The answer to your calculation is put in the GeoCraft Repository .
