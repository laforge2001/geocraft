GeoCraft Scientific Calculator

The GeoCraft scientific calculator performs scientific calculations (such as finding the cosine, arc tangent, or square root) on horizons loaded into GeoCraft.

GeoCraft Scientific Calculator: an Overview

The GeoCraft Scientific Calculator performs the selected calculation on the data selected for the b variable, and then writes the results of the calculation out as a new output variable.

Best Practices for Mathematical Operations in GeoCraft

When you perform a calculator function on two horizons, the two geometries must be identical. This means that each grid must contain an identical number of rows and columns, and the x and y locations of each node must be identical.

Note: The Basic Math, Boolean Algebra, Scientific, and Wavelet Math calculators now use the Free Form Horizon and Wavelet Calculators. Therefore, when you click the Open: Free Form 3D Equation Calculator (Horizon) button, the Free Form Calculator pops up with the equation already inserted in the Editor. This allows you to modify the equation before pressing Run in the Free Form calculator.

Using the GeoCraft Scientific Calculator

The following procedure provides a basic overview of how to use the GeoCraft Scientific Calculator. See each of the Scientific Calculator related topics to learn how each function works.

  1. Load horizon (grid) data into GeoCraft.
  2. Select a b variable.
  3. Select a c variable (the c variable is only necessary for the atan2 operation).

    4. Note: Variable c may be a number (for example, 2.5) instead of a horizon.

  4. Select the operation to perform:
    • atan2(b,c): Arc Tangent
    • cos(b): Cosine
    • sin(b): Sine
    • log10(b): Base 10 Logarithm
    • logn(b): Base e Logarithm
    • sqrt(b): Square Root
    • gmean(b): Grid Mean
    • gstdv(b): Grid Standard Deviation
    • gnorm(b): Grid Normalization
  5. Accept the default output name of a1, or type a new value in the field.
  6. Click Run the current algorithm.

When you click the Run the current algorithm button, GeoCraft runs the calculation in the background. The answer to your calculation is put in the GeoCraft Repository .
