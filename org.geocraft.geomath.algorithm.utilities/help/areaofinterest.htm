Create a 3D Area of Interest

Create an area of interest (AOI) for an existing 3D volume.

Overview of Create a 3D Area of Interest Algorithm

The Create Area of Interest algorithm allows you to create an area of interest (AOI) for an existing 3D volume in GeoCraft. You can then use the AOI in other calculations. This algorithm creates 3D AOIs; see Creating a 2D Area of Interest for instructions about creating 2D AOIs.

Note: 3D AOIs can also be created graphically in Map View by drawing a polygon around the area of interest. The method described here allows one to create either a rectangular AOI by limiting the Inline and Xline range, or a decimated volume AOI by setting a Delta or step, such as to process every 10th Inline or Xline for example.

Creating a 3D Area of Interest

  1. Load a 3D volume into GeoCraft.
  2. Select an Input Volume. GeoCraft populates the Input Volume Parameters and Output Area of Interest Parameter fields with values for the selected volume.
  3. Type values into the Output AOI Parameter fields to create your AOI:
    • Inline Start - Starting Inline number
    • Inline End - Ending Inline number
    • Inline Delta - Delta or step for processing inlines
    • xline Start - Starting Xline number
    • xline End - Ending Xline number
    • xline Delta - Delta or step for processing xlines

    Note: Start and End values are moved to the nearest valid Inline or Xline. If Delta is invalid, it is reset to the data value. For example if the first inline number is larger than the ending inline, because the line is reversed, then Delta will be a negative number. But if the user enters as a positive Delta value, it will be reset to the actual, negative, value.

    Hint: To see the actual values used to create the AOI, do not supply an Output AOI Name. The Start, End, and Delta values will be included in the default Output AOI Name.

  4. Optional Select the Input your own area of interest name? check box.
  5. Optional; only available if you selected the Input your own area of interest name? check box. Type in an Area of Interest Name.
  6. Click Run.

GeoCraft runs the calculation, and creates the AOI, storing it in the Repository, in the Areas of Interest folder. If you did not provide an output name, GeoCraft names the file varnnn=input volume name + process parameters.
