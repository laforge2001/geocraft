Horizon to Area of Interest

Create an area of interest (AOI) from an existing 3D horizon by finding the horizon edge.

Overview of Horizon to Area of Interest Algorithm

The Horizon to Area of Interest algorithm tries to create an area of interest (AOI) for an existing 3D horizon in GeoCraft by searching row by row for the first non-null node and the last non-null node. That is, it attempts to create the AOI as the outline of the edge of the input horizon.

Note: 3D AOIs can also be created graphically in Map View by drawing a polygon around the area of interest, and by creating a rectangular AOI by limiting the Inline and Xline range and setting a step or increment. The method described here is yet another way. Once created, the AOI is stored in the Repository. It can be dragged out onto a Map view for visualization.

WARNING: The 3D AOI is created "in memory". That is, it is not automatically saved and restored. Nor does a right-click on the AOI and selecting Save actually save anything! You must select Export and export the AOI to an ascii file. Then when you later restart GeoCraft you must right-click on the AOI folder in the Repository and select Load.

Running Horizon to Area of Interest

  1. Load a horizon into GeoCraft.
  2. Select an Input Horizon
  3. Accept the default output Area of Interest Name or enter a new name.
  4. Click Run.

GeoCraft runs the algorithm, creates the AOI, and storing it in the Repository, in the Areas of Interest folder.
