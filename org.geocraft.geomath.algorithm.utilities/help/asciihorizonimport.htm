ASCII Horizon Import

The ASCII Horizon Import algorithm imports one or more horizons in ASCII columnar format into GeoCraft. The input data format is expected to be something like X, Y, H1, H2, etc, where every horizon has the same X,Y grid nodes. Columns can be in any order. For each line in the ASCII file, the input data columns must be separated by spaces or tabs. The input horizon may be in time or depth but all output horizons must have the same index type, time or depth. Header lines at the top of the file may be skipped.

Input grid properties are not expected to be able to be read from the input data file. As a result, these must be supplied by the user, such as the grid origin, increment, angle, and the number of rows and columns.

Note: You can create up to 5 new horizons at once with this algorithm.

Converting ASCII Data Into Horizons

  1. Select the Input Directory where the ASCII file is located.
  2. Type in the Input file Name.
  3. Select the Index Type for the input data:
    • Depth: the horizon is in depth
    • Time: the horizon is in time
  4. Select the XY Units of the input data:
    • meter: the XY data in the ASCII data are in meters
    • foot: the XY data in the ASCII data are in feet
  5. Select the Data Units for the grid you are creating:
    • meter
    • foot
    • kilometer
  6. Type the Starting Line Number that is in the ASCII input file.
  7. Type the Number of Horizons that you want to create from the ASCII input file.
  8. Type the X Column #; the column number in the ASCII file that contains the X locations.
  9. Type the Y Column #; the column number in the ASCII file that contains the Y locations.
  10. Type the H1 Column #; the column number in the ASCII file that contains the actual horizon value. This field is repeated to specify the column number of each input horizon.

    11. Now, you will enter parameters for the output horizons.

  11. Type in the X Origin.
  12. Type in the Y Origin.
  13. Type in the Column Spacing.
  14. Type in the Row Spacing.
  15. Type in the # of columns (i.e. nodes in the X direction).
  16. Type in the # of rows (i.e. nodes in the Y direction).
  17. Type in the Primary angle.
  18. Type in the Null Value.
  19. Type the Hn Name. If you are creating multiple horizons, GeoCraft will display a name field for each horizon, which you can change if desired.
  20. Click Run.

GeoCraft stores each horizon in the Repository view, under the Grids folder. Use the GeoCraft Map Plot to view and examine results.
