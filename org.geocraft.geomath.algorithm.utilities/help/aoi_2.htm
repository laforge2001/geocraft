Create a 2D Area of Interest

Create an area of interest (AOI) for an existing line in a 2D volume.

Note: Although 3D AOIs can also be created graphically in Map View, 2D AOIs can only be created by running this module.

Overview of Create a 2D Area of Interest Algorithm

The Create Area of Interest algorithm allows you to create an area of interest (AOI) for an existing line in a 2D volume in GeoCraft. You can then use the AOI in other calculations. This algorithm creates 2D AOIs; see Creating a 3D Area of Interest for instructions about creating 3D AOIs. The Trace Decimation parameter allows users to skip every Nth trace. The default of 1 means process every trace.

Creating a 2D Area of Interest

  1. Load a 2D volume into GeoCraft.
  2. Select an Input 2D Volume. GeoCraft populates the Input 2D Volume Parameters with values for the selected 2D line.
  3. Type values into the Output AOI Parameter fields to create your AOI:
    • Shot Point Start - Starting shot point number
    • Shot Point End - Ending shot point number
    • Trace Decimation - Decimation factor for processing shot points: 1 means process every trace, 2 for every other

    Note: Start and End values are moved to the nearest valid shot point.

  4. Optional Select the Input your own area of interest name? check box.
  5. Optional; only available if you selected the Input your own area of interest name? check box. Type in an Area of Interest Name.
  6. Click Run.

GeoCraft runs the calculation, and creates the AOI, storing it in the Repository within the Areas of Interest folder. If you did not give an output file a name, GeoCraft names the file varnnn=input volume name + process parameters.
