Setting the AGC and Geometric Gain Parameters for Section View

Trace normalization for the Section Viewer is set in the Settings dialog. The Normalization option in the Rendering tab gives users several ways to set trace scales, such as by trace or display (absolute) average, trace or display maximum, or user defined limits. Trace average and maximum means the scale factor changes from trace to trace. Display average and maximum use a single scale value derived over all traces in the display.

However, sometimes the seismic gain decays with time or depth and these options are not enough. Other times one needs to equalize amplitudes between two or more volumes that have different scaling. For these cases an AGC or Geometric Gain control is necessary. These options are on the "AGC and Geometric Gain" tab in the Settings Dialog. The AGC option does not ask for a gain level. It sets the absolute average value over the moving window to 1.

  1. View a section plot.
  2. In the Layers tab, highlight the entity for which you want to change rendering settings.
  3. Right-click the mouse button on that entity.
  4. In the Settings dialog box, select the AGC and Geometric Gain tab.
  5. In the AGC and Geometric Gain tab, change settings for the following properties:
    • Click Apply AGC to apply automatic gain control for the display
    • Select the desired AGC Type window to use, either Boxcar or Gaussian
    • Enter an AGC Window Length or use the default
    • Click Apply Geometric Gain to remove spherical spreading effects
    • Enter an initial gain at zero time t0 or use the default
    • Enter the spreading constant n or use the default
    • Enter a time to limit the gain tmax or use the default
  6. Click OK or Apply
