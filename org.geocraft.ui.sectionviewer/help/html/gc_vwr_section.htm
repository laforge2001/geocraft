GeoCraft Section Plot

The Section Viewer provides visual representation for PreStack and PostStack 3D, and PostStack 2D seismic data.

GeoCraft Section View Settings

By default, the GeoCraft Map View inherits settings from previous plots that you created of the same type. Subsequent plots that you create in the Map View will be displayed with the same settings, except for the Minimum and Maximum Color Scale Settings, within the Map View.

Hint: See Viewing 2D Seismic Data in the Section View for special instructions that you must follow before you can view 2D seismic data in the section viewer.
