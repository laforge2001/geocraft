Manually Refreshing Data within the Section View

You can manually force the section view to redraw the section plot when you change data parameters.

  1. View a section plot.
  2. Change any data parameters, such as adding the Z range to the data viewed.
  3. Click the Redraw button on the toolbar.

GeoCraft redraws the section plot.
