Before You Use the GeoCraft Section View

This topic contains brief descriptions of important section plot features and functionality that you should understand before you use the viewer.

Feature or Function Explanation and Further Details
You must load data into GeoCraft before the View => Section Plot menu option activates If the repository does not contain any entities that the Section Plot can render, the View => Section Plot menu option will not be available. See Loading Data into GeoCraft.
Optional After GeoCraft displays the Section Plot, you may indicate whether data should be displayed along the inline or crossline See Using the Section Navigator to learn about setting an inline or crossline display for section plot data
View an inline, xline, or arbitrary Section line selected in Map View See Viewing Lines Selected in Map View within Section View for details
Data types you can view in the section plot

You can view the following data types in the Section plot:

  • Prestack 3D
  • Poststack 3D
  • Poststack 2D
2D seismic data must be viewed using special instructions You may use the same directions to view and analyze all other data types, but 2D seismic data must be viewed using different instructions. See Viewing 2D Seismic Data in the Section View for specific instructions.
Setting plot layout preferences Before you use any of the GeoCraft viewers, set the user preferences so that GeoCraft displays plotted data the way you want to see it. See
  • Setting Plot Layout Preferences.
