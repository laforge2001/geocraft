Plotting a Section View

  1. Load data into GeoCraft.
  2. Navigate to the data you want to view in the Repository, and right-click the entity you want to view.
  3. Select View=>Section View from the popup menu.
  4. Optional Use the Section Navigator on the toolbar to set different inline or xline values for the section view.
  5. View and analyze the data.
