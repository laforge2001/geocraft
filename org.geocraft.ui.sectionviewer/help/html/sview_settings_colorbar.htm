Setting the Color Bar Parameters for the Section View

Use the following instructions to set the color bar parameters for the Section View plot.

  1. View a section plot.
  2. In the Layers tab, highlight the entity for which you want to change rendering settings.
  3. Right-click the mouse button on that entity.
  4. In the Settings dialog box, make sure that the Color Bar tab is active.
  5. In the Color Bar tab, change settings for the following properties:
    • Colors => Select: select a different colormap to display data. By default, GeoCraft displays data in the Seismic colormap.
    • Colors => Open: open a colormap that is stored on your filesystem.
    • Edit => Reverse Colors: reverses the order of colors in the colormap.
    • Edit => Non-Editable: Makes the colormap non-editable until this option is turned off.
    • Edit => Circular Shift: Allows user to click and drag to rotate colors in the colormap.
    • Edit => Marker: Allows you to highlight a range of colors in the colormap by setting them to the Marker Color.
    • Start Value and End Value: type new values to set color ranges. NB: These values are active only when the Normalization option is set to "By Limits" in the Rendering tab.
    • Marker Color: click to select the Marker color.
  6. Click OK or Apply
