Viewing Lines Selected in Map View within the Section View

Section View can display Inlines, Xlines, or Arbitrary lines selected in Map View. You can broadcast the cursor position from the Map View to the Section View, or from the Section View to the Map View.

  1. Open Map View on the Survey node under 2D Seismic or 3D Seismic folders in the Repository tree. This node contains the geometry for the volume and must be displayed in Map View. If you also desire to view a particular horizon, simply open a Map View for that horizon and click the Cursor reception is disabled icon to enable cursor tracking. However, section lines can only be selected from the Survey Map View.
  2. Click the Cursor broadcast is disabled on the Survey Map View toolbar.
  3. If not already displayed, open Section View for the data volume.
  4. Click the Cursor receive is disabled on the Section View toolbar.
  5. In the Survey Map View Layers tab right-click on the Survey node and select either an Inline, Xline, or Arbitrary Section from the pop up menu.
  6. If you want to select a specific inline or xline, click the Readout tab in the Map View so you can read the inline and xline number.
  7. When the cursor changes to a plus sign over the map, select the line to display by pressing MB2. For arbitrary lines left click to draw the line and when done click MB2 to display the line in Section View. The selected line is now displayed in Section View.
  8. In Section View click the Cursor broadcast is disabled icon.
  9. In Map View click the Cursor reception is disabled icon.

Now as the cursor is moved around in Section View, its position can be seen in Map View - i.e. the Survey Map View or any other horizon displayed in Map View with cursor reception enabled. If the cursor is moved in Map View to intersect the line of section, its position in Section View is annotated by a plus sign at the top of the nearest trace.
