Viewing 2D Seismic Data in the Section View

The Section Viewer provides visual representation for PreStack and PostStack 3D, and PostStack 2D seismic data.

Note: Follow these instructions when you view 2D seismic data in the section view; otherwise, you will be unable to view 2D data. You must select the entire 2D project name in the Repository ; if you select individual 2D lines, you will not be able to view data in the section view.

To Plot 2D Seismic Data

  1. Load 2D seismic data into GeoCraft.
  2. In the GeoCraft Repository , right-click on the 2D project name.
  3. Select View => Section View.
  4. Click the Section Navigator button.
  5. Click Line, and select the 2D line you wish to display.

View and analyze the data in the section view.
