Unique Toolbar Buttons in the GeoCraft Section View

This table lists the toolbar buttons that are unique to the GeoCraft section plot, and explains how each button functions. See Common Features in GeoCraft Viewers for a description of the common toolbar buttons in the Section Plot and other GeoCraft viewers, and how those toolbar buttons function.

Name Icon How the icon works
Redraw redraws the section in the plot
Decrement decrements the inline/xline section in the plot
Increment increments the inline/xline section in the plot
Section Navigator opens the Section Navigator
Z Range opens the Z Range dialog box
Toggle Groups toggles section groups on and off
