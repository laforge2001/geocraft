Setting the Rendering Parameters for the Section View

Use the following instructions to set the rendering parameters for the Section View plot.

  1. View a section plot.
  2. In the Layerstab, highlight the entity for which you want to change rendering settings.
  3. Right-click the mouse button on that entity.
  4. In the Settings dialog box, make sure that the Rendering tab is active.
  5. In the Rendering tab, change settings for the following properties:
    • Select an Interpolation, either Linear or Stepwise
    • Select a Normalization option. NB: Color Bar start and end values are active only when the Normalization option is set to "By Limits".
    • Enter a Trace Exaggeration multiplier value
    • Enter a Trace Clipping value as a number of adjacent traces to limit the trace excursion
    • Enter a Percentile (0-50) value over all trace amplitudes from which to compute the scale factor
    • Enter a Transparency (0-100) value for this overlay entity. Values greater than 0 allow layers below the current entity to be visible.
    • Click Reverse Polarity to reverse trace polarity
    • Click Wiggle Trace to display the trace wiggle
    • Click Positive Color Fill to fill positive trace loops with the Positive Fill Color
    • Click Negative Color Fill to fill negative trace loops with the Negative Fill Color
    • Click Positive Density Fill to density fill positive trace loops
    • Click Negative Density Fill to density fill negative trace loops
    • Click Variable Density to turn on the variable density trace fill option
    • Click Wiggle Color to select the wiggle trace color
    • Click Positive Fill Color to select the positive trace color fill
    • Click Negative Fill Color to select the negative trace color fill
  6. Click OK or Apply
