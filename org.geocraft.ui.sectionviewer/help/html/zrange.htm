Setting Z Range Values in the Section View

You can select the Z range and set the Z range values within a section plot.

  1. View a section plot.
  2. Click the Z Range button on the toolbar.

    3. The Time/Depth Range dialog box for the Z range is displayed. By default, the maximum value for the Z range is displayed in the End Time field.

  3. Type in a Start Time, or use the scroll bar to set a value.
  4. Type in an End Time, or use the scroll bar to set a value.
  5. Click Ok.

GeoCraft redraws the section plot with the new Z range values.
