Using the Section Navigator

Use the Section Navigator to

  • set the resolution of the section image you are viewing,
  • indicate whether you want to automatically update the image upon inline or xline changes and
  • indicate whether the data should be plotted along the inline or crossline (xline) for a section plot.

You may also set a step length for the inline or crossline data that is plotted.

In the Section Navigator dialog box, the range of data values is displayed along a slider, from lowest to highest value. The Current field displays the lowest value by default. You can accept the default value, type a new value in the Current field, or move the slider bar to limit the range of data that you see in the Section Plot.

Plotting Data in the Section View

Tip: You must indicate whether the data should be plotted along the inline or crossline in the Section Navigator dialog so that GeoCraft can display data in the Section Plot.

  1. View a section plot.
  2. Click Open the Section Navigator on the toolbar.
  3. Optional Perform the following steps to set a different Resolution for the section view:
    1. Type a value in the Traces/Inch field to set a new resolution for traces per inch.
    2. Type a value in the Inches/Second field to set a new resolution for inches per second.
    3. Press Enter to apply the changes to resolution in the section view.
  4. Select or turn off the Auto-update on Inline or Xline changes checkbox. By default, this option is on.
  5. Select an option:
    • Inline: select this option to plot data along the inline data points
    • Xline: select this option to set plot the data by its crossline values
  6. Type a value in the Current field, or accept the default value.
  7. Optional Type in a Step length for the inline or xline value that you just set.
  8. Click Close
    9. .

GeoCraft closes the Section Navigator dialog box, and displays the data in the Section Plot.
