Toggling the Group Entities Display On and Off in the Section View

This topic is under development.

  1. Create a group of entities that you want to display together.
  2. Assign a group number to the entities that you have grouped together.
  3. Open the GeoCraft Section View.
  4. Click the Toggle Groups button on the toolbar.

  5. Select the group number that is assigned to the entities you want to view.
  6. View and analyze the data in the Section View.
