Changing the Title for a Scatter Plot

Use these instructions to change the title of a GeoCraft scatter plot.

  1. Perform the steps to view a scatter plot.
  2. Right-click the title of the scatter plot.
  3. In the Title Editor, type a new name in the Text: field.
  4. Optional Select a new Alignment for the title.
  5. Click OK.

GeoCraft displays the scatter plot with its new title.
