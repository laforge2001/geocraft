Changing the Scatter Plot from a Linear to a Log Scale

By default, GeoCraft plots scatter plot data on a linear scale. Use these instructions to change the scatter plot from a linear scale to a log scale.

  1. Perform the steps to view a scatter plot.
  2. Right-click on the X or Y axis of the scatter plot to display the Axis Editor.
  3. In the Scale group box, select the Log option.
  4. Click OK.

GeoCraft re-draws and displays the scatter plot on a log scale.
