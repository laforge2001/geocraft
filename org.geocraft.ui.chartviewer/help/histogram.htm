GeoCraft Histogram Viewer

The GeoCraft Histogram viewer displays a bar graph of the frequencies of different numerical values within a population. In GeoCraft, the population is a data type. You can render a histogram for the following data types in GeoCraft:

  • 2D survey lines
  • 3D PostStack seismic
  • horizons (grids)
  • well log plots

You can plot multiple data entities within a single histogram window to compare data frequencies. If you plot more than one entity, GeoCraft color codes each entity so that you can easily compare the data. You may also choose to data in individual plots.
