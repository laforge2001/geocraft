Setting the Z Bound Domains for a Scatter Plot Display

Use these instructions to set the Z bound domains for a Scatter plot in GeoCraft.

  1. Perform the steps to view a scatter plot.
  2. Click the Layer Model button to display the Layer Model and Readout tabs.
  3. With the Layer Model tab active, right-click on the names of the well logs and select Settings.
  4. Place your cursor in the Start Z field, and type in a new beginning value for the Z range.
  5. Place your cursor in the End Z field, and type in an ending value for the Z range.
  6. Click OK.

GeoCraft redraws the scatter plot with the new Z boundaries.
