Plotting Data in the Histogram Viewer

Use these instructions to generate a histogram chart for a single data entity.

  1. Load data into GeoCraft.
  2. In the Repository view, right-click the data that you want to analyze.
  3. Select Histogram from the popup menu.

GeoCraft generates the histogram view.
