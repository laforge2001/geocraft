Setting the Minimum (Start) and Maximum (End) Range in a Scatter Plot Display

Use these instructions to set the minimum (start) and maximum (end) range of display values for a GeoCraft scatter plot.

  1. Perform the steps to view a scatter plot.
  2. Right-click on the X or Y axis of the scatter plot to display the Axis Editor.
  3. Put your cursor in the Start: field, and type in the Start (minimum) for display values in the scatter plot.
  4. Put your cursor in the End: field, and type in the End (maximum) for display values in the scatter plot.
  5. Click OK.

GeoCraft re-draws and displays the scatter plot with its new start and end range values.
