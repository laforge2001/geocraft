Plotting Multiple Data Entities in the Histogram Viewer

Use these instructions to generate a histogram chart for a multiple data entities.

  1. Load data into GeoCraft.
  2. In the Repository view, right-click the first data entity that you want to analyze.
  3. Select Histogram from the popup menu.

    4. GeoCraft generates the histogram view.

  4. In the Repository view, highlight the next data entity that you want to analyze.
  5. Drag-and-drop the data entity into the existing histogram chart.

    6. GeoCraft generates a histogram chart for the second data entity. The second data entity is in a different color, so that you can compare the two data sets.

  6. Continue to drag-and-drop data entities into the existing histogram chart until you have added all of the data sets you want to compare.

View and analyze the data.
