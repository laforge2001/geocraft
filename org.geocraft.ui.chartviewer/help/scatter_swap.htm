Moving a Well Log from the X to the Y Axis on the Scatter Plot

Use these instructions to move well log data from the X to the Y axis on a scatter plot in GeoCraft.

  1. Perform the steps to view a scatter plot.
  2. Click the Layer Model button to display the Layer Model and Readout tabs.
  3. With the Layer Model tab active, right-click on the names of the well logs and select Swap Axes as shown in this figure.

GeoCraft swaps the well log data along axes. For example, if the Depth (DT) well log was plotted along the X axis, GeoCraft moves it to the Y axis.
