GeoCraft Scatter Plot View

Use the GeoCraft Scatter Plot to analyze and compare data from two different well logs in a crossplot view. The order of log selection does not matter. To reverse the axes, simply right-click on the plot entry under Wells folder in the Layers tab and select Swap Axes.

  1. Load data into GeoCraft.
  2. In the GeoCraft Repository , expand the Wells directory.
  3. Select a well, and expand the Logs directory underneath the well.
  4. Select two well logs inside the directory.
  5. From the menu commands, select View => Scatter Plot.
  6. To reverse axes, right-click on the plot in the Layers tab and select Swap Axes.
