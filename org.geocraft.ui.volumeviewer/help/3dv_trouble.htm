Troubleshooting GeoCraft 3D Viewer Problems

This topic provides symptom descriptions for some common GeoCraft 3D Viewer problems, and tells you how to resolve those problems.

This topic contains brief descriptions of important 3D viewer features and functionality that you should understand before you use the viewer.

Problem Possible Cause for Problem Solution
When I right-click in the Repository View, the Volume Plot pop-up menu does not display. You must load data that the 3D viewer can render into GeoCraft before the View => Volume Plot menu option activates If the repository does not contain any entities that the 3D viewer can render, the View => Volume Plot menu option will not be available. See Loading Data into GeoCraft.
The Volume Plot is taking a long time to render an object in 3 dimensions. You may have selected some options that will make the Volume Plot take longer to render larger objects. To speed up rendering, make sure that the following options are not selected in the Seismic 3D Settings dialog box:
  • Z Slice Display check box: Do not select this check box unless you are working with an extremely small volume; it can take several hours to render larger volumes in the Z slice plane
  • Enable smoothing interpolation check box
