Applying a Specific Colormap File to Data in the 3D View

By default, the GeoCraft 3D view draws all plots in the "seismic" colormap. However, you may also apply a specific colormap to data, instead of using the default seismic colormap provided by GeoCraft.

Note: GeoCraft provides several generic colormaps, in addition to the seismic colormap. If you want to change the generic colormap that is used to display data in the 3D view, see Changing the Colormap for a 3D View Plot.

  1. Perform the steps in Viewing an Object in the 3D Viewer to display an entity.

    2. Leave the Seismic 3D Settings dialog box open.

  2. Click the Attribute #1 tab.
  3. Choose Colors => Open, and select one of the following options:
    • Color File: load an existing color map
    • Generic (ASCII) Color File: load an existing generic ASCII color map
  4. Navigate to the file system and directory where the color map is located and select it.
  5. Click OK.

GeoCraft loads the colormap file, and the GeoCraft 3D View plots the volumes. Compare and analyze the data in the 3D view.
