Changing the Azimuth (Sunlight) Settings in the 3D Viewer

You can use the Azimuth settings to change orientation of the sun on an image that you are viewing in the GeoCraft 3D viewer. That is, this dialog box will change the direction from which the sun is shining down upon the objects you are viewing.

  1. View an image in the 3D viewer.
  2. Click Open the sun light settings.
  3. Use the following values to set the sun orientation; this is the side of the image on which the sun is shining:
    • north: 90
    • south: 270
    • east: 0 and 360
    • west 180
  4. Click OK.

View the image with its new azimuth settings.
