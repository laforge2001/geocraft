Changing the Zoom Speed in the 3D Viewer

The zooming speed works slightly different within the 3D viewer than it works in other GeoCraft viewers. Use the following steps to change the zoom speed within the 3D viewer.

  1. Perform the steps to load data into GeoCraft
    2. .
  2. View an object in the 3D viewer.
  3. Select an object in the 3D viewer on which you want to zoom in or out.
  4. Use one of these methods to control the rate of speed at which the 3D viewer zooms toward and away from objects:
    • To speed up the zoom rate, hold CTRL and move the mouse wheel
    • To slow down the zoom rate, hold SHIFT and move the mouse wheel
