Making an Object the Point of Rotation in the GeoCraft 3D Viewer

Perform the following steps to rotate a selected object in the GeoCraft 3D viewer window.

  1. Select the object that you want to rotate.
  2. Click the Anchor icon at the top left.
  3. Move the mouse over the view and click the left-mouse button.

This changes the view to move the point to the center. Now, the object will rotate around the point.

To reset the object to its original location, click the Home icon.
