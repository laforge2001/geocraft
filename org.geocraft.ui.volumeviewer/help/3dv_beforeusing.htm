Before You Use the GeoCraft 3D Viewer

This topic contains brief descriptions of important 3D viewer features and functionality that you should understand before you use the viewer.

Feature or Function Explanation and Further Details
Setting user (environmental) preferences in the 3D viewer See Setting Display Preferences for the GeoCraft 3D Volume Viewer
Data types you can view in the 3D viewer

You can view the following data types in the 3D viewer:

  • 2D and 3D horizons
  • 2D and 3D seismic data
  • culture data
  • faults
  • wells
You can have multiple instances of the 3D viewer open GeoCraft does not limit the number of instances of the 3D viewer that you have open; you can plot as many objects as you need to in separate plots.
You must load data into GeoCraft before the View => Volume Plot menu option activates If the repository does not contain any entities that the 3D viewer can render, the View => Volume Plot menu option will not be available. See Loading Data into GeoCraft.
