Changing the Colormap for a 3D View Plot

By default, the GeoCraft 3D View draws all plots in the Seismic colormap. However, GeoCraft provides several other generic colormaps that you may select when you are plotting data. This topic tells you how to apply a different colormap to data in the GeoCraft 3D view.

Note: If you have a specific colormap file that you apply to data, GeoCraft also provides a way to do that; see Applying a Specific Colormap File to Data in the 3D View.

  1. Perform the steps in Viewing an Object in the 3D Viewer to display an entity.

    2. Leave the Seismic 3D Settings dialog box open.

  2. Click the Attribute #1 tab.
  3. Choose Colors => Select => Colormap name.
  4. Click OK.

The GeoCraft 3D View plots the volumes. Compare and analyze the data in the 3D view.
