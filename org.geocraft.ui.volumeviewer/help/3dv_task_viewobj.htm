Viewing an Object in the 3D Viewer

Follow these instructions to view objects in the 3D viewer.

Caution!: Do not select the option to display Z slice data in the 3D View, unless you are working with an extremely small volume. It can take several hours to render Z slice data in the GeoCraft 3D View. If you do select this option, GeoCraft will display a warning message, and you must acknowledge the dialog prompt before GeoCraft begins rendering the plot.

  1. Load data into GeoCraft.
  2. In the Repository, expand the folder (for example, 3D Seismic that contains the data you want to view.
  3. Locate and select the object you want to view in the Repository.
  4. Right-click the object that you want to view, and select View => 3D View from the popup menu.

    5. The Seismic 3D Settings dialog will display, with the Navigation tab active.

  5. In the Inline group box, select the Display checkbox.
  6. Type in a Current value, or accept the value in the field.
  7. Type in a Step value, or accept the value in the field.
  8. In the Xline group box, select the Display checkbox.
  9. Type in a Current value, or accept the value in the field.
  10. Type in a Step value, or accept the value in the field.

    11. Caution!: Do not select the Display checkbox in the Z slice field, unless you are working with an extremely small volume. GeoCraft may take several hours to render the image if you select this option.

  11. Select the Display Labels check box to include display labels on the 3D view plot itself.
  12. Select the Enable Smoothing Interpolation check box to flatten the image.

GeoCraft renders the object in the 3D View. View and analyze the image.
