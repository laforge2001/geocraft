Comparing Two Volumes in a Single 3D View Plot

This task tells you how to compare 2 volumes within a single GeoCraft 3D plot.

  1. Perform the steps in Viewing an Object in the 3D Viewer to display the first volume.

    2. Leave the Seismic 3D Settings dialog box open.

  2. Click the Attribute #2 tab.
  3. Select the Volume check box.
  4. Select a Volume.
  5. Use the arrows to customize the Blend Amount, if necessary.

    6. By default, GeoCraft draws all plots with the "seismic" color map.

  6. To select a different color map for the comparison volume, choose Colors => Select => Colormap name.
  7. Type in a Start Value (or Min) for the volume, or accept the default.
  8. Type in an End Value (or Max) for the volume, or accept the default.
  9. Select a Marker Color for the volume.
  10. Select the Lock Range check box if you want for GeoCraft to compute the Min and Max values from all values in the data set. If you do not check Lock Range, GeoCraft computes Min and Max from the visible slices in the data set.
  11. Click OK.

The GeoCraft 3D View plots the volumes. Compare and analyze the data in the 3D view.
