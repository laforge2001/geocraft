Flooding Salt Velocity in GeoCraft: Starting the Procedure

Before you begin, ensure that the data you want to use meets all GeoCraft requirements for Velocity Flood data. You will always perform the steps documented in this topic, regardless of the type of salt flooding you perform. After you perform these steps, you will move on to a second task to complete salt flooding.

  1. Load ModSpec grid (horizon), PostStack 3D (volume), and optional Area of Interest data.
  2. Select a PostStack 3D input Volume.
  3. Optional; only perform this step if you want to use an Area of Interest in the calculation. Select the Use Area of Interest? check box.
  4. Optional Select an Area of Interest.
  5. Select a Flood Direction:
    • Flood Below: flood salt velocity below a horizon; you will select a Top Horizon below which to flood
    • Flood Above: flood salt velocity above a horizon; you will select a Base Horizon above which to flood
    • Flood Between: flood salt velocity between two horizons; you will select a Top Horizon and a Base Horizon between which to flood
  6. Depending upon your choice in the prior step, select the horizon(s) below, above, or between which you will flood salt.
  7. Select a Flood Type:
    • Constant: uses a constant value for flood velocity; follow the Flooding Salt Velocity with a Constant Flood Type instructions to complete velocity flooding
    • Gradient (V0 + g*z): uses a change in gradient (change in "rise over run") for flood velocity; follow the Flooding Salt Velocity with a Gradient Flood Type instructions to complete velocity flooding
    • Dataset: uses a volume for flood velocity; follow the Flooding Salt Velocity with a Dataset Flood Type instructions to complete velocity flooding
  8. Follow the specific instructions to flood salt velocity with a constant, gradient, or dataset flood type.
