Velocity Smoothing

The Velocity Smoothing algorithm removes rough transitions from a velocity model by smoothing adjacent values. An Area of Interest can be used to reduce the amount of processing. This module may be used to smooth other types of volumes in addition to velocity volumes.

Caution: Although this module can be used to smooth any volume, if the volume contains zeros, using the Slowness option will result in division by zero. Therefore, do not apply this in the Slowness domain with seismic volumes.

Overview of Velocity Smoothing

Velocity Smoothing averages adjacent values based upon the following options:

  • kernel width
  • smoothing kernel type
  • smoothing domain
  • width (and amplitude, if applicable) of smoothing kernel

Smoothing a Velocity Volume

  1. Load the velocity volume data into GeoCraft.
  2. Select the input Volume.
  3. Optional; only perform this step if you want to use an area of interest. Select the Use Area of Interest? check box.
  4. OptionalSelect an Area of Interest.
  5. Accept the values in the Kernel Width (Indexed Values) X, Y, Z fields, or type in new values.
  6. Click Recalculate Dimensions.

    7. GeoCraft returns the kernel width dimensions for the volume you selected.

  7. Select a Smoothing Kernel Type:
    • Boxcar - performs the calculation using a rectangular window function; the amplitude parameter is calculated at full value, and the width parameter is calculated as a percentage
    • Gaussian - performs the calculation using a Gaussian function; parameters are always calculated full width by half maximum value
  8. Select a Smoothing Domain:
    • Slowness - smooth the volume using the reciprocal of the phase (apparent) velocity
    • Velocity - smooth the volume using apparent velocity
  9. The items displayed in the Parameters group box vary depending upon the Smoothing Kernel Type you selected:
    • If you selected Boxcar, type values in the Boxcar Amplitude and Boxcar Width (percentage) fields.
    • If you selected Gaussian, type a value in the Full Width at Half Maximum field.
  10. Accept the Output Volume Name, or type a new name in the field.
  11. Click Run.

GeoCraft runs the algorithm, and the new volume populates in the Repository under the Seismic folder.

You may now use GeoCraft plotting and viewing tools to view the volume.
