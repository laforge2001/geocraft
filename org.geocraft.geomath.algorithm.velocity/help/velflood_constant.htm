Flooding Salt Velocity with a Constant Flood Type

  1. Select the Constant(s) or Horizon(s) option:
    • Use a constant - GeoCraft uses a constant value to calculate flood velocity. GeoCraft contains a default value in the Flood Velocity (ft/s) field. Water has a constant velocity of 5000 feet per second, or approximately 1500 meters per second. However, if you have data about the water temperature, depth, or pressure, you may want to type a different value in the Flood Velocity (ft/s) field.
    • Use a horizon - GeoCraft varies the flood velocity with a horizon. If you use a horizon for a variable velocity, select a Velocity horizon (ft/s).

    Note: The data units of each horizon that you select during this task must match the data unit of the input volume. For example, if the input volume is measured in feet per second, all horizons used in this calculation must also be measured in feet per second.
    If the data units for the input volume and any horizon in the calculation do not match, GeoCraft displays an error in the System Messages window and does not complete the calculation.

  2. In the Volume Name (ft/s) field accept the default value, type characters after the underscore to append to the volume name when the calculation runs, or clear the field and type in a new name.
  3. Click Run.

GeoCraft creates a new volume that is flooded with the salt velocity parameters you set during this task. You can use a GeoCraft viewer to review the results of the calculation.
