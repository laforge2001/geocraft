Horizon Stretch - Time-Depth Conversion

Horizon Stretch converts horizons from time to depth or from depth to time.

Horizon Stretch Overview

Horizon Stretch converts time-based horizons to depth or depth-based horizons to time. This algorithm uses a velocity volume to perform the conversion, and can convert multiple horizons at once. The velocity volume used to control stretch can be either a time or a depth volume, independent of the horizon being converted.

Note:Horizon Stretch interpolates velocity at each node on the horizon grid.

Caution: When loading data, make sure that the unit labels are correct for the input velocity volume and the input horizons. This is easily done by viewing the volume along with the horizons and seeing that they are in the same area.

Converting Horizons from Time-to-Depth or Depth-to-Time

  1. Load the input horizons and the velocity volume into GeoCraft.
  2. Click the Add button next to the Horizons field.

    3. A Selection Dialog displays, listing all of the horizons that you have loaded into GeoCraft.

  3. Select the horizons you want to use in the calculation and click OK.
  4. In the Horizons field, select individual horizons and use the Up and Down buttons to move the horizons toward the top or bottom of the calculation order. If you selected a horizon you do not want, highlight the horizon and click Remove.
  5. (Optional) Select the Use Area of Interest check box and select an Area of Interest.
  6. Select a Velocity Volume.
  7. Select a Conversion Method:
    • Cell-based - vertical velocities are held constant between input values
    • Knee-based - vertical velocities are linearly interpolated between input values
  8. In the Output Grid Suffix field, accept the default or type a new value in the field. This value gets appended to the end of the original input horizon name when GeoCraft creates the new, converted horizon.

    9. Hint: Consider a label to annotate the output domain. If the input is in time, try "dep", or if the input is in depth, try "tim".

  9. Click Run.

A progress bar is displayed while the conversions run. When the jobs finish, the converted horizons are saved in the Repository view, under the Horizons folder.

You may now use the GeoCraft to view the converted horizons.
