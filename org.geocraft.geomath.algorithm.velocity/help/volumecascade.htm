Volume Cascade (d->t->d)

Volume Cascade converts volumes from depth to time using one velocity, then from time to depth using a second velocity. The intermediate time volume is output with an _d2t extension.

Volume Cascade (d->t->d) Overview

The Volume Cascade algorithm uses velocity volumes to perform depth-to-time followed by time-to-depth conversions on volumes. You may convert several volumes at once, and can use an Area of Interest in the calculation to limit the processing area.

Cascading a Volume from Depth to Time followed by Time to Depth

  1. Load volume data into GeoCraft.
  2. In the Depth Volumes field, click Add.
  3. In the Selection Dialog, highlight the depth horizons you want to use in the calculation and click OK.
  4. In the Depth Horizons field, select individual horizons and click Up and Down to sort the volumes in the calculation order.
  5. Select an Interval Velocity Volume (V1) to use to convert volumes from depth to time.
  6. Select an Interval Velocity Volume (V2) to use to convert volumes from time to depth.
  7. Optional; only perform this step if you want to use an Area of Interest in the calculation. Select the Use Area of Interest? check box.
  8. Optional Select an Area of Interest.
  9. Select a Conversion Method:
    • Cell-based - vertical velocities are held constant between input values
    • Knee-based - vertical velocities are linearly interpolated between input values
  10. Accept the default in the Output Depth to Time Volume Suffix field, or type in a new value.
  11. Accept the default in the Output Time to Depth Volume Suffix field, or type in a new value.
  12. Accept the default Time Sample Rate, or type a new value in milliseconds.
  13. Accept the default Depth Sample Rate, or type a new value in feet or meters.
  14. Accept the default Maximum Time, or type a new value in milliseconds.
  15. Accept the default Maximum Depth, or type a new value in feet or meters.
  16. Click Run.

A progress bar is displayed while the conversions run. When the job completes, load the cascaded volume(s) into the Repository window.

You may now plot and view the data in one of the GeoCraft viewers.
