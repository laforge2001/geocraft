Flooding Intervals with a Gradient Flood Type

Interval velocity flooding using a gradient applies an equation of the form V0 + G*(z-Z0), where Z0 is the reference depth corresponding to initial velocity V0. Hence the input volume must be a depth velocity volume. The V0 and G parameters may be constants or horizons (grids) that vary laterally.

Note: Data units for each constant or horizon must match the data unit of the input velocity volume. If units for the input volume and a horizon do not match, GeoCraft displays an error and does not run the calculation.

  1. Select the Constant(s) or Horizon(s) option:
    • Use a constant - Set a constant velocity V0 value to flood the depth interval in the Flood Velocity(V0) field and set a constant value in the Flood Gradient(G) field.
    • Use a horizon - Select a Velocity(V0) Grid and a Gradient(G) Grid.
  2. Select a Reference Depth or Horizon:
    3. A reference depth specifies the depth (Z0) at which V0 applies in V0+G*(z-Z0). For example, suppose a V0 value of 9300 f/s was derived over an interval from 10,000 to 12,000 ft, then in order to make V0 be 9300 at the top of the interval, the reference depth must be 10000 (i.e. V = 9300 + G*(z-10000)).
    • Use a constant - GeoCraft uses a constant value to calculate the reference depth. Type a value in the Reference(Z0) Depth field.
    • Use a grid - GeoCraft varies the reference depth with a depth horizon. If you use a horizon to vary reference depth, select a Reference(Z0) Grid.
  3. In the Output Volume Name field accept the default value or clear the field and type in a new name.
  4. Click Run.

GeoCraft creates a new volume that is flooded with the velocity parameters set during this task. You can use a GeoCraft viewer to review the results of the calculation.
