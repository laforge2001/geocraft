Multi-Flooding Velocity Preview

Multi-Flooding Velocity Preview allows you to perform a quick visual quality assurance check of horizons and parameters used in a constant velocity flood between multiple pairs of horizons at once. This is done by flooding only the current Inline and Xline in the 3D Viewer. When Run is pressed, the output volume is put on the Repository and must be displayed in the 3D Viewer. As you navigate through the volume, only the current inline and xline are processed and results are sent back to the viewer. When results are ok, the entire volume can be processed and output.

Overview of the Multi-Flooding Preview Algorithm

Multi-Flooding Preview performs a velocity flood with a constant velocity between several pairs of horizons, allowing you to quickly check flood results. This algorithm also differs from the standard Velocity Flood by allowing you to import several pairs of horizons and do a constant velocity flood between each pair of horizons in one pass.

Note: When you run this algorithm, the output volume must be displayed in the 3D viewer. Whenever the Inline or Xline are changed, the algorithm is automatically re-run to process the new lines and the 3D Viewer is automatically updated. When the results are satisfactory, the entire volume can be processed by right-clicking on the output volume in the Repository, selecting Export, and specifying a data store.

Performing a Quality Assurance Check on Data Sets with the Multi-Flooding Preview Algorithm

Caution: When loading data, make sure that the unit labels are correct and consistent for all input volumes and horizons. This is easily done by viewing the volume along with the horizons and seeing that they are in the same area.

  1. Load volume data into GeoCraft.
  2. Load horizon data into GeoCraft.
  3. Select an input Volume.
  4. Click Add to select your top horizons.
  5. In the selection dialog, highlight the horizons you want to use in the preview and click OK.
  6. Optional In the Top Horizons list box, highlight a horizon and click the button to Remove it from the calculation; or to move it Up or Down in the multiples calculation order.
  7. Repeat steps 4-6 to select and order the Base Horizons for the multiples flooding preview.
  8. Accept the Flood Velocity #n, or type a new value in the field. Each pair of top and base horizons will have its own field to set flood velocity. This field defaults to 5000 feet per second.
  9. Accept the Volume Name, or type a new value in the field. This is the name GeoCraft gives to the new volume that results from this calculation.
  10. Click Run.

GeoCraft performs the calculation. A process bar displays while the algorithm runs.

When the algorithm completes its calculation, the Volume view opens to display the preview data.
