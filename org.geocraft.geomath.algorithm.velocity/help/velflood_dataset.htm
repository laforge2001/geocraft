Flooding Salt Velocity with a Dataset Flood Type

  1. Select a Dataset volume.

    2. Note: The data units for each volume that you select during this task must match the data unit of the input volume. For example, if the input volume is measured in feet per second, all volumes used in this calculation must also be measured in feet per second.
    If the data units for the input volume and any volume in the calculation do not match, GeoCraft displays an error in the System Messages window and does not complete the calculation.

  2. In the Volume Name (ft/s) field accept the default value, type characters after the underscore to append to the volume name when the calculation runs, or clear the field and type in a new name.
  3. Click Run.

GeoCraft creates a new volume that is flooded with the salt velocity parameters you set during this task. You can use a GeoCraft viewer to review the results of the calculation.
