Volume Stretch

Volume Stretch converts time volumes to depth, or depth volumes to time. The velocity volume used to control stretch can be either a time or a depth volume, independent of the volume being converted.

Volume Stretch Algorithm Overview

Volume Stretch converts time volumes to depth, and depth volumes to time. An Area of Interest (AOI) may be used to limit processing.

Note: If you select an Area of Interest for the calculation, the algorithm only converts traces from within the AOI. Traces outside the AOI are set null.

CAUTION!: Some 2D data stores limit the number of characters in file names. For example, in some data stores the file name must be 9 characters or less. If you are working with 2D volume data from a data store with file name limitations, modify the GeoCraft Volume Name to comply with the data store requirements.

Converting a Depth or Time based Volume

  1. Load the depth or time seismic volume into GeoCraft.
  2. Load the depth or time velocity volume into GeoCraft.
  3. Select the Type of Volume:
    • 2D: two-dimensional volume data
    • 3D: three-dimensional volume data
  4. Click Seismic Volume, and select the input volume to convert.
  5. Click Velocity Volume (ft/s), and select a velocity volume.
    6. Note: The velocity volume used to control stretch can be either a time or a depth volume, independent of the volume being converted.
  6. Optional Select the Use Area of Interest (AOI)? check box if desired.
  7. Optional Select an Area of Interest. This field is only available when the Use Area of Interest (AOI)? check box is selected.
  8. Select an option in the Parameters field:
    • Cell-based - vertical velocities are held constant between input values
    • Knee-based - vertical velocities are linearly interpolated between input values
  9. Enter a valid value in the Sample Rate field in time or depth.
  10. Enter a valid value in the Maximum Time or Depth field.
  11. In the Output Volume Name field accept the default value, or type a new value.
  12. Click Run.

A progress bar is displayed while the conversion runs. When the job completes, the converted volume is in the Repository.

You may now plot and view the data in a GeoCraft viewer.
