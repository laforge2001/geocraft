Loading ABAVO Input Data

You must select the 2D or 3D input data and pre-processing elements before you can calculate an ABAVO output series. Intercept/gradient data is required if you want to run an amplitude with variation offset (AVO) analysis. You can create intercept/gradient data, from input near and far stacks.

  1. Load volume data into GeoCraft.
  2. Optional Load wavelet and horizon data into GeoCraft, if you want to use wavelets and horizons in your processing options.
  3. From the main menu, click Perspectives and select either ABAVO 2D or ABAVO 3D
  4. In the Input Data section, select a Data Mode:
    • All Data: runs the calculation based on all data in the data set.
    • Peaks and Troughs: runs the calculation based on only the peaks and troughs in the data set.
  5. Select Volume A.
  6. Select Volume B
  7. Optional; only select if you want to convert near/far data into an intercept/gradient data format. Select the Calculate Pseudo Intercept and Gradient from Near/Far check box if your input is a near offset and a far-offset stacked section.
  8. Type a value in the Volume A Scalar field, or accept the value in the field.
  9. Type a value in the Intercept Scalar field, or accept the value in the field. You can type a value in this field if you are converting near/far data into intercept/gradient data.
  10. Optional; if you want to use a wavelet to filter Volume A data in your calculation, then Select the Apply Wavelet Filter? check box.
  11. Optional Select a Wavelet Filter for Volume A.
  12. Type a value in the Volume B Scalar field, or accept the value in the field.
  13. Type a value in the Gradient Scalar field, or accept the value in the field. You can type a value in this field if you are converting near/far data into intercept/gradient data.
  14. Optional; if you want to use a wavelet to filter Volume B data in your calculation, then Select the Apply Wavelet Filter? check box.
  15. Optional Select a Wavelet Filter for Volume B.
  16. In the A/B Alignment Options area, you select pre-processing options. To change any of the values in the Auto-Align B to A field, select the Auto-Align B to A check box and type a new value in the following fields:
    • Correlation Window-size of the correlation window to use for alignment; units are in milliseconds
    • Maximum Shift-maximum trace shift; units are in milliseconds
    • Correlation Threshold-minimum required correlation to use for alignment; the threshold limits are from 0 to 1
    • Smoothing Filter Length-size of the smoothing filter to use for alignment; units are in milliseconds
    • Near Amplitude (Ampl.) Threshold-near amplitude threshold to use for alignment; this unit is a percentage
    • Amplitude (Ampl.) Threshold Window-amplitude threshold limit to use for alignment; units are in milliseconds

Next you must enter parameters in the Processing Bounds panel.
