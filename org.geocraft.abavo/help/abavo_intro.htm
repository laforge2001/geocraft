Introduction to ABAVO

ABAVO (Amplitude/Background and Amplitude Variation with Offset) is an amplitude versus offset crossplot analysis program developed by ConocoPhillips Company.

ABAVO analyzes amplitude variations with offset to determine whether lithology or fluid changes in a geographical area might be visible. ABAVO uses either normal incidence (A) and gradient (B); or near and far seismic sections as the input source of data for crossplot analysis. Then, geoscientists interpret these A and B crossplots. The corresponding fluid factor trends are calculated by ABAVO and displayed graphically.

Overview of ABAVO

ABAVO can analyze 2D and 3D seismic data. This is the ABAVO workflow; follow these steps when working in ABAVO:

  1. Compute the intercept and gradient for a background data series.
  2. Compute the intercept and gradient for an anomalous data series.
  3. ABAVO uses the intercept-gradient crossplots to construct a background ellipse and a maximum ellipse; this results in a defined ellipse model where AVO classes are color-coded by region.
  4. You may push seismic data sets through the ellipse model to locate and analyze other anomalies by color.
  5. Optional You may also construct a polygon from the crossplot and use the polygon to locate similar anomalies on the seismic data set.

Starting ABAVO Algorithms

The best way to start up ABAVO is to first open the ABAVO perspective if GeoCraft is not already in this perspective. Do this by clicking Perspectives => ABAVO 2D for 2D seismic, or Perspectives => ABAVO 3D for 3D seismic. When GeoCraft switches to the ABAVO perspective, the ABAVO Crossplot window is created and the Crossplot A vs B algorithm is opened in the main window. Other ABAVO algorithms can be launched directly from the crossplot window using the ABAVO Algorithm Buttons.
