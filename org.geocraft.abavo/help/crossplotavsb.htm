Generating an A vs. B Crossplot

You can use ABAVO to generate A versus B crossplots when you want to compare two data sets (for example, you may want to compare background and event data sets).

Overview of the Process to Generate an A vs. B Crossplot

Perform the following tasks, in order, to generate an A vs. B crossplot:

  1. Load input data into ABAVO.
  2. Select processing bounds for the data that ABAVO will use in the calculation.
  3. Run the calculation.
  4. View the A vs. B data in the ABAVO crossplot.
