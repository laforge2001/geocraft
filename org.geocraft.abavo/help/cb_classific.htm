Entering ABAVO Classification Data

The Classification and Output Data sections specify the ABAVO class background generation method and output file.

  1. Select a Classification Method:
    • Class of Sample selects a data set based on an individual sample.
    • Class of Peak/Trough selects a data set based on the peaks and troughs from a sample classification.
  2. Select a Region Definition.
    • Ellipses selects the current Ellipse Model.
    • Polygons selects the current Polygon Model.
  3. Accept the default in the Output Class Volume Name field, or type another name.
  4. Select either an 8 or 16-bit integer Class Volume Type
  5. Click Run.

You can now view color class volume in the Section or 3D viewers.
