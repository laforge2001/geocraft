Loading a Polygon Model

You can use the ABAVO Crossplot window to create new polygon models based upon ABAVO calculations that you just ran, or you can load existing polygon models into ABAVO.

Overview of Polygon Regions

Polygon regions determine the class background color for a model. Then the class background color is shaded between the background ellipse and the maximum ellipse. The background color shading is lighter near the background ellipse, and turns a deeper shade towards the maximum ellipse. For example, all class 3 data is green. Darker background shades indicate stronger class events. If the ABAVO Crossplot shows a dark green reflection on a crossplot, you are looking at a strong class 3 event.

Save polygon regions models that you use frequently in the GeoCraft repository. Then, when you need to manipulate or analyze the data, you can load the polygon regions models into the ABAVO Crossplot window, instead of having to re-create the calculation each time you need to view the results of the data set.

You must have the ABAVO Crossplot window open to work with polygons regions models in GeoCraft. Click this button to open the Polygon Regions Model dialog box.

Using ABAVO to Work with an Existing Polygon Model

  1. Open the Polygon Regions Model dialog box.
  2. Click Load.
  3. Navigate to the directory where the polygon region model is stored.
  4. Highlight the model you want to load.
  5. Click OK.
    The polygon regions model is loaded into the ABAVO Crossplot. You can view, manipulate, and analyze the data in the model.
