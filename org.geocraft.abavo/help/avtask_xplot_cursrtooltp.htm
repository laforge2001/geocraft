Tracking the Intercept and Gradient Values with Cursor Movements in the ABAVO Crossplot View

Click to toggle the Cursor tooltip on and off.

When the Cursor tooltip is enabled, hover the mouse anywhere on the ABAVO Crossplot. A tooltip displays the Intercept (A) and Gradient (B) values of the cursor's current location. As you move the mouse, the tooltip upates the A and B values with the current cursor location.
