Selecting Processing Bounds for ABAVO Data

This section discusses setting processing limits for creating the ABAVO Crossplot. It also covers limiting processing for generating the Class volume and the A+B volume. Processing is primarily limited by either restricting the inline or the xline range of the input volume. Limiting the inline and/or xline range reduces the number of points displayed in the crossplot. If over 200,000 points are displayed, the crossplot may take several seconds to display. In order to focus on a single horizon, you must also set up time or depth ranges for both the background series and the anomaly series. These windows can be either constant times or depths, or they can be horizon based.

  1. You can limit the number of points to crossplot by either selecting an AOI, or by specifying inline and xline ranges directly.
  2. Optional Select an Area of Interest.
  3. If you did not select an AOI as part of the processing boundary, accept the default Inline and Xline ranges or type new Start and End values into any of the following fields:
    • Start Inline
    • End Inline
    • Start Xline
    • End Xline
  4. Select a Time Mode for each data series. The option you select will dictate the fields that are available for data input.
    • Between Times or Depths: if you select this option you will type values in the Start Time (Depth) and End Time (Depth) fields.
    • Relative to Horizon: if you select this option you must specify a Top Horizon and type values in the Relative Start and Relative End fields, such as -20 and 20 to begin 20 ms above the horizon and end 20 ms after the horizon. If input is in depth, these values are in the preferred depth unit.
    • Relative to Horizons: if you select this option you must specify a Top Grid and Base Grid, and then type values in the Relative Start and Relative End fields, such as -20 and 20 to begin 20 ms above the horizon and end 20 ms after the horizon. If input is in depth, these values are in the preferred depth unit.

You are now ready to press Run. Please refer to the ABAVO Tutorial for information regarding selecting AVO class regions.
