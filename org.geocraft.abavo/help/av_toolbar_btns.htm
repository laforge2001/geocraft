ABAVO Crossplot Toolbar Buttons

This table provides a quick reference for each toolbar button that is available in the ABAVO Crossplot. The buttons are listed from the upper-most row to the lowest row, in left-to-right order.

Tip: Hover the cursor over any toolbar button to see its function.

Toolbar Button Functionality
Zoom In moves toward an object, magnifying it; click this button to zoom in by the current zoom factor of the crossplot; zooming in can also be done by rolling the mouse wheel
Zoom Out moves away from an object, allowing you to see the object in a larger perspective; click this button to zoom out by the current zoom factor of the crossplot; zooming out can also be done by rolling the mouse wheel
Pan mode is disabled

toggles the pan mode on and off within the display; perform these steps to use this feature:

  1. Click Pan mode.
  2. Click the mouse button anywhere in the Crossplot window.
  3. Hold the mouse button down and drag the cursor in any direction within the crossplot. ABAVO tracks your movements, scrolling the crossplot window in the same direction as the mouse cursor.
  4. Click Pan mode once more to turn off this feature when you are done using it.
Home View returns the object of focus in the crossplot view to the position in which it was originally mapped
Cursor broadcast toggles the display of the cursor location on and off; when this feature is on, a light-yellow plus sign tracks your mouse movements on-screen, and the X, Y, points are displayed on the bottom of the screen. Move the mouse to broadcast the current cursor location. Hover the mouse over an area to display a tooltip with the current intercept and gradient point where the cursor is sitting.
Layer model is shown toggles the Display List on and off
Edit the crossplot layout launches a dialog box where you set crossplot attributes. You can specify title height; label height and width; and axis height and width.
Cursor tooltip is disabled toggles the cursor tooltip display on and off; when this feature is on, the intercept and gradient points of the cursor are displayed in a tooltip format as the mouse hovers over items on-screen
Communication is disabled toggles communication on and off; when communication is activated, points chosen using the selection polygon or selection ellipse will be broadcast to external applications
Edit the ellipse regions model launches a dialog box that allows you to edit the property settings for the ellipse regions model on which you are working in the crossplot
Edit the polygon regions model launches a dialog box so that you can edit the property settings for the polygon model on which you are working in the crossplot
Edit the depth color bar allows you to edit the colors associated with the depth values of the crossplot points
Edit the class color bar allows you to edit the colors associated with the polygon model
Crossplot anchor toggles the crossplot anchor on or off; when the anchor is on, the center of a plot will be fixed to 0,0 as point series are added
Tip: If you are trying to crossplot values that are all positive or all negative, then the anchor should be turned off, so that the center of the plot is not anchored to 0,0.
Construct the background ellipse allows you to draw the points for a background ellipse
Construct the maximum ellipse allows you to draw the points for a maximum ellipse
Construct the selection ellipse allows you to draw the points for a selection ellipse
Data series drop-down list click the drop-down arrow and select which series to use for data input for the job you are running
Major axis drop-down list click the drop-down arrow and select which major axis to use for the regression line in an A+B calculation
Plot A vs. B algorithm launches the Crossplot A vs B algorithm
Class Volume launches the Class Volume algorithm
