Entering ABAVO A+B Data

The A+B module calculates an A+B volume. The B-section scalar and offset values are taken from the selected regression line or they may be specified manually.

  1. Type in a B-Section Scalar value; this indicates the spatial and time-dependence of a disturbance that can propagate as a wave.
  2. Type in a B-Section Offset value; this value accounts for displacement.
  3. Accept the A+B Volume Name, or type a new, unique name in the field.
  4. Select an A+B Volume Type:
    • 8-bit integer: the A+B calculation will create a volume that is stored in 8-bit integer data format
    • 16-bit integer: the A+B calculation will create a volume that is stored in 16-bit integer data format
    • 32-bit float: the A+B calculation will create a volume that is stored in 32-bit floating-point data format
  5. Click Run.

View and manipulate the results in the ABAVO Crossplot.
