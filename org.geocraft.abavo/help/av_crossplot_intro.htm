Introduction to ABAVO Crossplot

After ABAVO calculates A and B, you can use the ABAVO Crossplot (shown here) to view the results. You can use the ABAVO Crossplot to compare data series and to construct ellipses and polygons.

Tip: Darker background (class) color shades on the crossplot indicate anomalies. The darker a background color is shaded, the more anomalous the data points are.

ABAVO Crossplot View window

Opening the ABAVO Crossplot View

By default, the Crossplot view displays when you are in the ABAVO perspective. However, if the Crossplot window is not displayed, perform either of these steps to open the window:

  • If you are in the ABAVO perspective, select Perspectives=>Reset Default Layout.
  • Select Window=>ABAVO Crossplot.
