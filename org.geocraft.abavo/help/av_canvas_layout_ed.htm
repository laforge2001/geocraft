Changing the Appearance of Text, Titles, and Labels in a Crossplot

You can change ABAVO crossplot attributes to emphasize or remove items on a polygon or ellipse regions model.

Editing ABAVO Crossplot Attributes

Use this procedure to change the appearance of a crossplot:

  1. From the ABAVO Crossplot window, click Edit crossplot layout.
  2. Optional Clear the Title Visible check box to hide the crossplot title.
  3. OptionalType a value in the Title Height field. The value in this field is in points.
  4. OptionalIn the Axes section, type a value in each of the axis width and height fields. The value in these fields is in points.
  5. OptionalIn the Labels section, type a value in each of the label width and height fields. The value in these fields is in points.
  6. Click OK.
