Loading an Ellipse Regions Model

You can use the ABAVO Crossplot window to create new ellipse regions models based upon calculations that you have just created in ABAVO, or you can load existing ellipse regions models into ABAVO.

Overview of the Ellipse Regions Model

Save ellipse regions models that you use frequently in GeoCraft. Then, when you need to manipulate or analyze the data, you can load the ellipse regions models into the ABAVO Crossplot window, instead of having to re-create the calculation each time you need to view the results of the data set.

You must have the ABAVO Crossplot window open to work with ellipse regions models in GeoCraft. Click this button to open the Ellipse Regions Model dialog box.

Using ABAVO to Work with an Existing Ellipse Regions Model

  1. Open the Ellipse Regions Model dialog box.
  2. Click Load.
  3. Navigate to the directory where the ellipse region model is stored.
  4. Highlight the model you want to load.
  5. Click OK.
    The ellipse regions model is loaded into the ABAVO Crossplot. You can view, manipulate, and analyze the data in the model.
