Generating an A+B Volume

This topic provides links to the tasks you need to complete to generate an A+B volume.

Note: By default, data output from an A+B process is in 32-bit format.

Overview of the Process to Generate an A+B Volume

Perform the following tasks, in order, to generate an A+b volume:

  1. Load input data into ABAVO.
  2. Select processing bounds for the data that ABAVO will use in the calculation.
  3. Select classification parameters for the ABAVO class background output file.
  4. Run the calculation.
  5. View the data sets in the ABAVO crossplot.
