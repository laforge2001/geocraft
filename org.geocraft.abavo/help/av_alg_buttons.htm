ABAVO Algorithm Buttons

In the ABAVO Crossplot view, you can launch ABAVO algorithms directly using the three buttons above the crossplot grid:

  • Crossplot A vs B launches the ABAVO Crossplot algorithm for either 2D or 3D seismic. See Generating an A vs. B Crossplot for detailed instructions about creating an A vs B crossplot.
  • Class Volume launches the Class Volume algorithm. See Generating a Class Background Volume for detailed instructions about creating a class background volume.
  • A+B Volume launches the algorithm to compute A+B. See Generating an A+B Volume for detailed instructions about creating an A+B volume.
