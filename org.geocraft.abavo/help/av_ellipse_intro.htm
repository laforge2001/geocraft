ABAVO Ellipse Regions Model

You can use the ABAVO Crossplot view to build new ellipse models based upon ABAVO calculations you have just run, or you can load existing ellipse models into the viewer for interpretation.

Overview of the Ellipse Regions Model

You must have the ABAVO Crossplot window open to work with the ellipse regions model. The following links contain information about using features of the ellipse regions model:
