Constructing Ellipse Models

Use the following procedure to construct a background, maximum, or event ellipse model within the crossplot view.

Caution: You must use the same Active Data Series number and regression line to create the background and event ellipses. If you use a different Active Data Series number or regression line for either ellipse, the color (background) class does not display.

  1. Select the Series # that contains the data you want to plot.
  2. From the Display List, right-click one of the following ellipse models and select its corresponding popup menu:
    1. Background Ellipse => Construct the background ellipse
    2. Maximum Ellipse => Construct the maximum ellipse
    3. Selection Ellipse => Construct the selection ellipse
  3. Position the cursor in the crossplot. The mouse pointer turns into a hand, indicating that ABAVO is ready to construct the ellipse model.
  4. Drag the mouse along the crossplot to set the boundaries of the model. Ensure that all data points are captured within the ellipse when you are drawing the background ellipse model.
  5. Left-click the mouse to set the ellipse boundaries.
