Changing Line Position in a Crossplot

This topic provides instructions about how to change the position of lines in an ABAVO crossplot.

Overview of Changing Line Position in an ABAVO Crossplot

If you are working in an ABAVO crossplot and the Class Background or Region Bounds do not display, it is usually because some of the region lines are crossed. After you resolve this problem, the Class Background and Region Bounds will display in the crossplot.

Changing the Position of Lines within an ABAVO Crossplot

Use these instructions to change the position of a line in the Crossplot window.

  1. Hold down the SHIFT key and left-click the small black box at the end of a line to select it.

    2. Tip: When a line is selected, the small black box changes to a large black box.

  2. Continue holding the SHIFT key and drag the selected line to its new position in the Crossplot window.
  3. Click mouse button 3 (MB3) to refresh the plot.
