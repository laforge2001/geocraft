Changing the Class (Background) Colors in an ABAVO Crossplot

You have two choices when you want to change the class (background) colors in an ABAVO crossplot. You can apply a new colormap to the crossplot, or you can apply custom colors to the crossplot.

Note: When you make color class changes for an ABAVO crossplot, you are only making changes for that crossplot; the changes that you make will not be retained and applied to future crossplots that you create.

Applying a New Colormap to an ABAVO Class

  1. Click Edit the depth color bar.
  2. Select File => Open => Load Colormap Option.
  3. Click OK.

Applying Custom Colors to an ABAVO Class

  1. Click Edit the depth color bar.
  2. Click Marker Color.
  3. Set the custom color via one of the following options:
    • rotate the color wheel until you see the color you want to use
    • type a hexidecimal value in the Color Name field
    • set a color using the spinner buttons for Hue, Saturation, Value, Red, Green, and Blue
    • select a color from the Pallete
  4. Click OK twice to set the custom color, and apply it to the crossplot.
