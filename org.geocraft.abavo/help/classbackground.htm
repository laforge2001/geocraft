Generating a Class (Background) Volume

The class color (or background color) displays AVO classes as a color volume. Each class of data is assigned a different color, and the colors range in shade from light to dark. A darker shade indicates a stronger class of data (i.e. farther from the background trend). For example, all class 3 data is green. However dark green indicates crossplot a strong class 3 anomaly. Use ABAVO to generate ABAVO class backgrounds.

Overview of the Process to Generate a Class Background Volume

Perform the following tasks, in order, to generate a class background volume:

  1. Load input data into ABAVO.
  2. Select processing bounds for the data that ABAVO will use in the calculation.
  3. Select classification parameters for the ABAVO class background output file.
  4. Run the calculation.
  5. View the data sets in the ABAVO crossplot.
