Creating Polygon Models

Polygons determine the class colors that are used in models. Class colors are used to shade the area in between the background and maximum ellipses in a model. When you are viewing an ellipse model, darker hues (shades) of a class color indicate that a stronger event is being represented in the data set. You may create up to 64 polygons within an ABAVO Crossplot view.

  1. Click Edit the polygon regions model to open the Polygon Regions Model dialog box.
  2. Optional Select the Fill Polygons check box if you want the polygons filled in with color on the crossplot.
  3. Accept the value in the Normalization Factor field, or type in a new value.
  4. Optional Turn off the check boxes for any polygons that you want to create but do not want to display in the crossplot.
  5. Click Create for a polygon that you want to build in the crossplot.
  6. Click OK.
  7. Left-click a location in the crossplot window to start building the polygon.
  8. Left-click other locations in the crossplot window to set the boundaries for the polygon.
  9. Hold SHIFT and click MB3 to create the final boundary point on the crossplot and close the polygon.
  10. Repeat the prior steps in this procedure for each polygon you want to draw on the crossplot.

Tip: If you want to save this table of polygons and reuse it, click Save on the Polygon Regions Model dialog box. When prompted, provide a file name. ABAVO gives the file *.polygon_model as an extension.
