Major Axis Drop-down List

Use this feature to set the major axis for the regression line from which you want to obtain the B-section Scalar and B Offset values for an output A+B operation. Perform the following steps to set the major axis:

  1. Click Perpendicular (this is the default major axis).
  2. Select one of the following options:
    • Perpendicular—minimizes the perpendicular distance from each point to the line of regression.
    • Least Squares (LSQ)—approximates a data set so that the sum of the squares of the distances from the observed points to the curve is a minimum.
    • Reduced Mean (RMA)—approximates a data set using the Reduced Mean Average (RMA) of the points in the data set.
  3. Set any other ABAVO Crossplot features that you need to use, and create the crossplot.
