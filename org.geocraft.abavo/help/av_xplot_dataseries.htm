ABAVO Active Data Series

ABAVO stores up to 10 data sets at a time in the Active Data Series feature. You can overwrite any data set. Use the following procedure when you are working with ABAVO polygons and ellipses.

  1. Open the Crossplot view.
  2. Perform one of the following steps:
    • To retain the data set currently associated with the Series number that is displaying in the field, select a new Series from the list.
    • To overwrite the existing data set currently associated with the Series number that is displaying in the field, leave the Series number displaying. ABAVO will display a warning dialog stating that you are about to overwrite existing data. Acknowledge that you want to overwrite the existing data.

    Caution: If you do not change the Active Data Series number before ABAVO plots the data points, you will lose existing data.

  3. Plot the calculation.
