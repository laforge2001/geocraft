Changing Ellipse Models

Use the following procedure to modify an existing background, maximum, or event ellipse model within the crossplot view.

Caution: You must use the same Active Data Series number and regression line to create the background and event ellipses. If you use a different Active Data Series number or regression line for either ellipse, the color (background) class does not display.

  1. Select a new Series # to hold the data you want to plot.
  2. From the crossplot view, click Edit the ellipse regions model.
  3. In the Ellipse Regions Model dialog, click Load.
  4. Navigate to the ellipse model that you want to modify.
  5. Select the file (it will be named filename.ellipse_model) and click OK.
  6. From the Display List, right-click the ellipse you want to change, and select Construct the name ellipse.
  7. Position the cursor in the crossplot. The mouse pointer turns into a hand, indicating that ABAVO is ready to construct the ellipse model.
  8. Drag the mouse along the crossplot to change the boundaries of the model.
  9. Left-click the mouse to set each ellipse boundary point.
  10. Click MB3 to set the final boundary point of the ellipse model.
