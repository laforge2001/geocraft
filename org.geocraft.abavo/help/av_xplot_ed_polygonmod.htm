Editing a Polygon Regions Model

Use this procedure to modify an existing polygon regions model in ABAVO.

  1. Select a new Series # for the data you want to plot.
  2. From the crossplot view, click Edit the polygon regions model.
  3. In the Polygon Regions Model dialog box, click Load.
  4. Navigate to the polygon that you want to modify.
  5. Select the file (it will be named filename.polygon_model) and click OK.
  6. From the Display List, right-click Selection Polygon and select Construct the selection polygon.
  7. Position the cursor in the crossplot. The mouse pointer turns into a hand, indicating that ABAVO is ready to modify the polygon model.
  8. Drag the mouse along the crossplot to change the boundaries of the model.
  9. Left-click the mouse to set each polygon boundary point.
  10. Click MB3 to set the final boundary point of the polygon model.
