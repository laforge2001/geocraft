GeoCraft Concepts and Sessions

This section of the help provides conceptual information regarding GeoCraft and its Sessions that will help you use GeoCraft to its full potential. Basically, GeoCraft is a framework for rapid development and deployment of algorithms to help resolve problems in the geoscience domain. From a users perspective GeoCraft is a home for geoscience algorithms that help them to do their job better or more efficiently. GeoCraft comprises:

  • a general purpose geoscience development platform
  • the ABAVO perspective, for performing Amplitude Variation with Offset (AVO) analysis
  • the GeoCraft default perspective called GeoMath Algorithms, that contains geoscience algorithms
  • simple visualization and data exploration tools

GeoCraft is an open source project developed by ConocoPhillips Company; see the GeoCraft opensource web site for more details.
