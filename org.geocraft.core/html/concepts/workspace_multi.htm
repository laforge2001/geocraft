Creating Multiple GeoCraft Workspaces

You can create as many Geocraft workspaces as you need, to separate and save your project data. For example, you may want to create separate workspaces for:

  • project data sets in a specific unit system (meters or feet)
  • project data sets that are grouped by geographical location
  • project data sets that you are working on for a specific client

If you create multiple workspaces, when you launch GeoCraft, you will be prompted to select the workspace in which you want to work.

Creating a New GeoCraft Workspace

To create a new GeoCraft workspace, launch GeoCraft. You will be prompted with a Geocraft Workspace Chooser dialog:

Type a new value into the Workspace field, to create a new workspace.

Note: Each workspace must have a unique name.
