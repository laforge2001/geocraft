Saving a GeoCraft Session

Warning! If you click the X in the top right corner of the GeoCraft window to either exit or quit the session, this immediately closes all separate plot windows. As a result, any plots in separate windows cannot be saved as part of your session state. Instead, when you are ready to exit GeoCraft, always use File => Exit on the main menu.

There are several ways that you can save the state a GeoCraft session to a session file (*.gcs file extension):

  1. Every time you exit GeoCraft, you are prompted by the Save Session dialog to save the session. Click Yes, and you will be prompted in a Session Selection dialog for a filename and directory into which to session file.
  2. Select File => Save Session at any time to save the session in which you are currently working. GeoCraft saves your session in its current state to the active session file.
  3. Select File => Save Session As to save your current session with a new name. Use this option when you have made changes to the current session (for example: changing the default layout for GeoCraft; or changing the project data that is being used)

    4. When you select the File => Save Session As option, GeoCraft prompts you to provide a name for the file. You may place the session file in the workspace Sessions subdirectory or browse to another directory.

  4. Select File => Save Session As Batch to save your current session as a batch process.
