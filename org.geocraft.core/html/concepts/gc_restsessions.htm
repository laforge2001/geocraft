Switching GeoCraft Sessions

You can switch from one GeoCraft Session (project) to another without having to restart GeoCraft.

  1. Select File > Restore Session.

    A Current Session dialog box is displayed, prompting you to save the current session. Click Yes or No

  2. Afterwards the Session Selection dialog is displayed, allowing you to choose which session you want to restore.
  3. The dialog starts in the workspace's sessions folder. If your session file is elsewhere, navigate to its directory.
  4. Select the session to which you want to switch.
  5. Click OK.

GeoCraft closes the session in which you are currently working, and loads the session that you selected.
