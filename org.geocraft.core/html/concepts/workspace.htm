The GeoCraft Workspace

The GeoCraft Workspace is a directory generally located in your /home area. GeoCraft stores the following types of information in the Workspace:

  • session files, which have a *.gcs filename extension and are stored in a subdirectory named /sessions
  • batch session files, which have a *.gcb filename extension and are stored in a subdirectory named /gcbatch
  • a hidden file containing your last active session and the last option selection to restore it on launch
  • global user preferences
Session files can be copied from one workspace to another as long as you want to restore the same data and have file access permission for that data store.

Closing GeoCraft Sessions

Warning! If you click the X in the top right corner of a GeoCraft window to quit the session, this immediately closes all separate plot windows. As a result, any plots in separate windows cannot be saved as part of your session state. Instead, when you are ready to exit GeoCraft, always use File => Exit on the main menu.
