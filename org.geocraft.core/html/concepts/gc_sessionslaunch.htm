How Session Files Behave When Launching and Closing GeoCraft

A workspace contains saved sessions, including a memory of the last active session file you used and how you started that particular session. If you save or restore a GeoCraft session, that becomes your last active session. The next time you launch GeoCraft, that session will be displayed in the dialog box for your selection.

How Session Files Behave When You Launch GeoCraft

When you launch GeoCraft, a Geocraft Workspace Selection dialog displays:

GeoCraft remembers and restores all settings on this dialog box to their settings the last time you launched GeoCraft. In particular, the restore option is set depending on whether the check box was selected or cleared the last time you used it.

Starting a Different GeoCraft Session

When GeoCraft launches, you have the option to restore the last active session (if one exists). If you prefer to restore a different session, perform the following steps:

  1. Launch GeoCraft.
  2. When the Geocraft Workspace Selection dialog is displayed, do not select your last active session.
  3. When GeoCraft is launched, select File => Restore Session.
  4. Follow the directions in the Switching GeoCraft Sessions topic to select the session in which you want to work.

How Session Files Behave When You Close GeoCraft

When you select File => Exit, the Save Session dialog box is always displayed.

You have the following options on this dialog:

  • Yes: opens a Session Selection dialog that prompts you for a session filename. You may save the session file in the workspace sessions subdirectory or browse to another directory.

    • Caution: If you type in a name for an existing sessions file, you will overwrite the file.

  • No: closes GeoCraft without saving any changes to the session file
  • Cancel: leaves GeoCraft open
