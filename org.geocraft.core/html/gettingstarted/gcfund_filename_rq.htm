GeoCraft File-naming Requirements

All GeoCraft file names must have a unique name, regardless of where the file is stored. For example, two different files in separate project directories cannot share the same name. Every file in the GeoCraft repository must have a unique name to work properly in GeoCraft.

Note: If you try to save a file in GeoCraft, and the repository already contains a file with that name, GeoCraft displays a small yellow triangle warning icon next to the output name with a message. If you ignore the warning message and continue with the calculation, you will overwrite the existing file.
