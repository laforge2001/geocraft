Properties View

To display properties for an entity, select the data entity in the Repository view. The Properties view populates with properties-related data for the entity.

Some entity properties can be edited in the Properties view, such as horizon Onset Type. To edit a property value, right click in the Value column for the property and select Edit. If the property cannot be edited, you will receive a warning. Otherwise, you will be presented with a list of allowed values to select from.
