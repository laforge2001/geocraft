GeoCraft Viewers

You can use the following GeoCraft tools to view and analyze data:

  • volume (3D) viewer
  • histogram viewer
  • map (plot) viewer
  • scatter plot viewer
  • section viewer
  • table viewer
  • wavelet viewer

Note: Well logs may be viewed in the map, scatter plot, section, or volume viewers.
