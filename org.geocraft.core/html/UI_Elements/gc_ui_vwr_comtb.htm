GeoCraft Viewers: Common Toolbar Buttons

The following table provides a description of common GeoCraft viewer toolbar icons and describes how to use them.

Name Icon How the icon works
Zoom in magnifies an object, making it larger
Zoom out moves away from an object, so you can see more perspective on the object
Zoom window hold down the SHIFT key and zoom in on a specific region of the object in the viewer window
Pan click Pan, and then click and drag the object to pan to a new location on the object
Color change the background color of the viewer
Home view return a selected object to its original position within the viewer
Cursor broadcast is disabled toggles the cursor location on and off. When the cursor broadcast is on, the current X, Y, location of the mouse cursor display at the bottom of the graph.
Cursor receive is disabled toggles the cursor reception on or off
Layer model is shown/hidden toggles the Layers and Readout tabs so that they are displayed or hidden within the viewer
Cursor tooltip is disabled toggles the cursor tooltip on or off; when the cursor tooltip is on, a tooltip displays the X,Y coordinates when the mouse pointer is moving across an object in the viewer
Help launches context-sensitive help for the viewer
