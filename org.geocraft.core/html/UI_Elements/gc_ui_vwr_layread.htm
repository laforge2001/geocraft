GeoCraft Viewer Layer Model and Readout Tabs

Each viewer displays the layer model and readouts in tabs to the right of the graphic by default. Use the following tips when working with the layer model and readout information:

  1. To free up more room on-screen, toggle the Layer Model is Shown button off. When you need to work with the layer model and readout tabs displayed again, click Layer Model is Shown.
  2. The Layer Model shows all of the objects that the viewer is capable of displaying. You can work with objects in the display list by using the following hints:
  • Turn objects off one at a time by clearing the check boxes next to each object name.
  • Expand and collapse object hierarchies by clicking on the key icons on the left side of hierarchical view.
