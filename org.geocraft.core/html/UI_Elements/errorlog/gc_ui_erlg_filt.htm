Filtering Events in the GeoCraft Error Log View

Use the Log Filters dialog box to configure the information that displays in the GeoCraft Error Log.

  1. Click View Menu.
  2. Select Filters from the popup menu.
  3. Turn on (select) the check boxes in the Event Types group box to indicate which events you want to monitor in the Error Log.
  4. Optional Turn off the check box for the Limit visible events to: field if you do not want to limit the number of visible events in the Error Log.
  5. Optional Type a value in the Limit visible events to: field to change the number of visible events in the Error Log.
  6. Select an option for Show events logged during:
    • All sessions: data is stored in the Error Log for all sessions until you clear the Workspace
    • Most recent session: data from the most recent session is stored in the Error Log
  7. Optional Select Enable stack filters to hide stack trace elements.
  8. Optional Click Add.
  9. Type the string for which you want to filter and click OK.
  10. Click OK.
