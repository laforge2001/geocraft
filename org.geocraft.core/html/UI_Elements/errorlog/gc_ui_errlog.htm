GeoCraft Error Log View

The GeoCraft Error Log displays data about the operation of the program, such as information, warnings, and errors.

Overview: GeoCraft Error Log View

By default, the Error Log view is activated when a new event message is triggered in GeoCraft.

The Error Log is in the bottom-right quadrant of the interface as a tabbed view. Click the Error Log tab to make it the active view.

This image captures part of the GeoCraft Error Log, with the features you will use most often called out. These features are in the upper-right corner of the Error Log.
