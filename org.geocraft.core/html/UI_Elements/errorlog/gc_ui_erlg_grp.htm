Grouping Events by Type in the GeoCraft Error Log

You can set the GeoCraft Error Log to display events by

  • Session: all logged events from a GeoCraft session are listed together;
  • Plug-in: all logged events from one GeoCraft plug-in are listed together; or
  • None: all logged events in GeoCraft are listed sequentially by date.

Setting the Error Log to Group Events by Type

  1. Click View Menu.
  2. Select Group By from the popup menu.
  3. Select the Group By option (Session, Plug-in, or None) that you want to use for sorting GeoCraft events.

The Error Log sorts events by the new grouping.
