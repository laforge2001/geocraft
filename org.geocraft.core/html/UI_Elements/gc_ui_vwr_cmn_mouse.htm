Common Mouse Controls in the GeoCraft Viewers

The following table provides a description of the mouse controls available for most GeoCraft viewers.

Action to Perform Mouse Control to UseIcon
Zoom toward an object

scroll down with the mouse wheel

If you do not have a mouse wheel, click the right and left mouse buttons and drag the mouse downward to zoom in on an object
Zoom away from an object

scroll up with the mouse wheel

If you do not have a mouse wheel, click the right and left mouse buttons and drag the mouse upward to zoom away from an object
Zoom horizontally on an object*

Hold CTRL and scroll the mouse wheel

If you do not have a mouse wheel, hold CTRL, click the right and left mouse buttons and drag the mouse upward to zoom away from an object
Zoom vertically on an object*

Hold SHIFT and scroll the mouse wheel

If you do not have a mouse wheel, click the right and left mouse buttons, hold SHIFT, and drag the mouse upward to zoom away from an object

* The 3D viewer performs different actions when the CTRL + mouse wheel and SHIFT + mouse wheel interactions are used. See Changing the Zoom Speed in the 3D Viewer for more details.
