Algorithms

GeoCraft is designed so that geophysical researchers can easily add new algorithms into the framework without having to create data input/output (I/O) and visualization tools.

Algorithms are grouped by function in the algorithm tree.

Before you can run an algorithm, you must load data into GeoCraft.

Tip: To quickly locate an algorithm, type the first few letters of its name in the type filter text field, located directly underneath the Algorithms tab on the interface.
