Restoring Closed Windows

When you are working in GeoCraft, you can move and close windows. If you cannot find one of the following windows, you may have accidentally closed it while you were working in the application:

  • 3D Viewer
  • ABAVO Crossplot
  • Algorithms
  • Error Log
  • Help
  • Map Viewer
  • Properties
  • Repository
  • Section Viewer
  • Wavelet Viewer

Restoring a Closed Window

Select Window =>Window Name to restore a closed window.
