Saving GeoCraft Sessions

When you run GeoCraft, the current session can be saved in any of the following ways:

  1. When you exit the program, GeoCraft automatically prompts you to save the session if there have been changes to the session state since the last time the session was saved. Click Yes to save the session changes.
  2. By selecting File => Save Session from the menu. This creates a regular session file, with a *.gcs filename extension.
  3. By selecting File => Save Session as Batch from the menu. This creates a batch session file, with a *.gcb filename extension.
  4. By selecting File => Restore Session. This allows you to switch to a different session file. GeoCraft saves the session file in which you are currently working, and allows you to select a new session file in which to work.
