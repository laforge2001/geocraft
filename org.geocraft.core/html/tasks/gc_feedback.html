Providing Feedback to the GeoCraft Team

The GeoCraft development team wants to meet the needs of your Business Unit. Please help us do that by providing feedback.

Within GeoCraft, select Help => SendFeedbacktoHPCGroupatCOP. This menu command launches a web form you can use to:

  • make suggestions and requests for new features
  • report unexpected results when you run a calculation
  • request additional functionality or features
  • provide suggestions for improvements to the user interface and usability
  • report missing or inaccurate online documentation
  • request additional (or more in-depth) documentation
  • suggest documentation improvements or changes
  • provide usability feedback to improve the online documentation
  • tell us anything else that is on your mind
