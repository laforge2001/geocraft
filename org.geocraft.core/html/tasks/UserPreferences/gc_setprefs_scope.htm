Setting the Scope of GeoCraft Preferences

In GeoCraft, you can set preferences on a project-specific or a global basis for some properties. By default, preferences are set globally for all properties. GeoCraft does this by storing project specific settings in the current workspace and global settings in an internal master workspace. If you select the "Enable Project-Specific Settings" button when you set user preferences, these are saved to the current workspace.

Global vs. Project-specific Settings

  • Global: settings apply this property to all projects; this is the default setting for all preference properties in GeoCraft
  • Project-specific: settings apply only to the current workspace; project-specific settings always override global settings

You can select a global or project-specific preference for these properties:

  • 3D Viewer
  • ABAVO Crossplot
  • Plot Layout
  • Wavelet Plots

Setting the Scope of Preferences for a Property

Note: If you are going to enable project-specific settings, you must be in the correct workspace before you change preferences.

  1. Optional; only perform this step if you are enabling project-specific settings and are in the wrong workspace. Select File => Workspace => Other.
  2. Select Edit => Preferences.
  3. Select the property which you want to set (for example, Plot layout).
  4. Select the Preference Scope option that you want:
    • Enable Global Settings
    • Enable Project-Specific Settings
    • Click OK to close the dialog box.
