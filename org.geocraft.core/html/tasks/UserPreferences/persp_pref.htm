Setting Perspectives Preferences

GeoCraft has two main perspectives: ABAVO and a default called GeoMath Algorithms. This topic tells you how to set GeoCraft perspective behaviors, and how to specify a GeoCraft perspective.

Setting Default Behavior for GeoCraft Perspectives

Use this procedure to configure how GeoCraft opens new perspectives (in the same window or a new window) and new views (within the existing perspective, or as a fast view).

  1. Select Edit => Preferences.
  2. Select Perspectives.
  3. Set the Open a new perspective option.
  4. Set the Open a new view option.
  5. Click OK.

Setting a Default GeoCraft Perspective

Use this procedure to set a default perspective in GeoCraft.

  1. Select Edit => Preferences.
  2. Select Perspectives.
  3. In the Available Perspectives field, highlight the perspective (ABAVO or GeoCraft) that you use the most.
  4. Click Make Default.
  5. Click OK.
