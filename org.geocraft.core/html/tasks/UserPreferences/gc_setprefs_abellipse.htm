Setting ABAVO Ellipses Preferences

You can view the background, maximum, and selection ellipse at one time on the ABAVO crossplot. This topic shows you how to configure the Ellipse settings so that each ellipse model displays uniquely in the crossplot. If you perform this procedure, you can easily distinguish each ellipse in the ABAVO Crossplot.

  1. Select Edit => Preferences.
  2. Expand ABAVO Crossplot and select Ellipses.
  3. Use the controls on the dialog box to configure the following properties for each ellipse:
    • Background
    • Maximum
    • Selection
  4. Click OK.
