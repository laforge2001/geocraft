Setting ABAVO Crossplot Preferences

Use the following procedure to change how GeoCraft displays data in the ABAVO Crossplot. You may change the settings for all properties or for a single property in the ABAVO Crossplot within this dialog box.

  1. Select Edit => Preferences.
  2. Select ABAVO Crossplot.
  3. Use the controls on the dialog box to configure the following properties for the ABAVO Crossplot:
    • plot background color
    • model background color
    • axis line settings
    • plot title settings
    • axis label settings
    • axis range settings
  4. Click OK.
