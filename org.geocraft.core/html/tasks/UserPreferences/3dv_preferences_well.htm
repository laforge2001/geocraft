Setting Well Display Preferences in the GeoCraft 3D Volume Viewer

  1. Select Edit=>Preferences.
  2. Select 3D Viewer to expand the list of options.
  3. Type a value in the Log Excursion field. The value you type in the Log Excursion field must be between 1-1000.
  4. Type a value in the Pick Radius field. The value you type in the Pick Radius field must be between 1-1000.
  5. Click Apply.
