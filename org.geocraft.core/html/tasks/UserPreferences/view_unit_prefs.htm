Viewing Unit Preferences for a Workspace Session

When you create a new workspace in GeoCraft, the GeoCraft Session Chooser dialog prompts you to set unit preferences for the workspace. After you have set unit preferences for a workspace, you cannot change them; you may only view them.

  1. Select Edit => Preferences.
  2. Select Unit Preferences to view the current unit preferences.
  3. Click OK to close the dialog box.
