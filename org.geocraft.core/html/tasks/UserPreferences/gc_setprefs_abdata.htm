Setting ABAVO Data Series Preferences

You can run and view multiple operations on the same crossplot in ABAVO if you use the Data Series feature. This topic shows you how to configure Data Series settings so that each Data Series displays uniquely in the crossplot. If you perform this procedure, you can easily distinguish the data from different operations in the ABAVO Crossplot.

  1. Select Edit => Preferences.
  2. Expand ABAVO Crossplot and select Data Series.
  3. Use the controls on the dialog box to configure the following properties for each data series (1 through 10) in the ABAVO Crossplot:
    • color
    • size: thickness of the lines in pixels
    • symbol: shape of the symbols along the line (for example, a circle, a cross, or an X)
  4. Click OK.
