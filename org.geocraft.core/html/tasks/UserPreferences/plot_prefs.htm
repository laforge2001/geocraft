Setting Plot Layout Preferences

Use this procedure to configure how GeoCraft creates and displays plots in your environment.

  1. Select Edit => Preferences.
  2. Select Plot Layout.
  3. Select the preference scope.
  4. Select a Create plots in option:
    • Active window creates a new window that is docked inside of GeoCraft for plot data.
    • A separate window launches a floating window, separate of the GeoCraft user interface, for plot data.
  5. Click OK.
