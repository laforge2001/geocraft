Restoring GeoCraft Default Preferences

Use the following procedure to restore the default GeoCraft preferences for any set of properties:

  1. Select Edit => Preferences.
  2. Select the property for which you want to restore the default preferences (for example, Plot layout).
  3. Click Restore Defaults, and click OK to close the dialog box.
