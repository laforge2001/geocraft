Setting Horizon Intersection Display Preferences in the GeoCraft 3D Volume Viewer

  1. Select Edit=>Preferences.
  2. Select 3D Viewer to expand the list of options.
  3. Select Horizon Intersection.
  4. Type a value in the Width field to set thickness for the line where the volume slice and horizon intersect. The Width value must be between 1 and 40.
  5. Click Apply.
