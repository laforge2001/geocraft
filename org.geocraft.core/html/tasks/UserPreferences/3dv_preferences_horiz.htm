Setting Horizon Display Preferences in the GeoCraft 3D Volume Viewer

  1. Select Edit=>Preferences.
  2. Select 3D Viewer to expand the list of options.
  3. Select Horizon.
  4. Select the Enabled check box to decimate horizon data with triangles. This selection systematically resamples data in a triangular pattern to reduce the number of samples that are used to represent the data.
  5. Select a resampling Type:
    • Automatic: GeoCraft uses a predetermined algorithm to decimate the horizon data
    • Manual: GeoCraft uses a numerical factor that you provide to decimate the horizon data
  6. Optional. Type in a Factor value.
  7. Click Apply.
