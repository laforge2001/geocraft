Setting Unit Preferences

When you launch GeoCraft, the GeoCraft Session Chooser prompts you to specify a workspace session. If you create a new workspace, you must specify a new unit system for that workspace.

Setting Unit Preferences for a Session

Note: You may only set the unit system for a session once, when you create that session. After the unit system is set for a session, it cannot be changed.

  1. Select the Time Units for the session.
  2. Select the X,Y Unit for the session.
  3. Select the Depth for the session.
  4. Click OK to set all unit systems for the session.
