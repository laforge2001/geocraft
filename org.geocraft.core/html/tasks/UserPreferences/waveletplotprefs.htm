Setting Wavelet Plot Preferences

Use this procedure to configure the functionality of the GeoCraft Wavelet plot viewer.

  1. Select Edit => Preferences.
  2. Select Wavelet Plots.
  3. Select the preference scope.
  4. In the Time Domain group box, select a scaling option.
    • Automatic scaling: GeoCraft automatically applies a scale factor to the time domain
    • Manual scaling: GeoCraft uses a factor that you supply to scale the time domain
  5. Optional; only perform this step if you set the Time Domain to use Manual scaling. Type in a Scale Factor.
  6. In the Amplitude Spectrum group box, type in a Maximum Frequency or accept the value.
  7. Select a scaling option.
    • Automatic scaling: GeoCraft automatically scales the frequency spectrum.
    • Manual scaling: GeoCraft uses a factor that you supply to scale the frequency spectrum
  8. Optional; only perform this step if you set the Amplitude Spectrum to use Manual scaling. Type in a Scale Factor.
  9. Set a Spectrum Type for wavelet plots:
    • Amplitude: wavelets plot in linear amplitude as a function of frequency
    • Linear Power: wavelets plot in linear power (amplitude squared)
    • Decibel Power: wavelets plot in a decibel (db) power scale
  10. If Decibel Power is selected, enter a dB Range, or accept the default value.
  11. In the Phase Spectrum group box, select the Display the Phase Spectrum Plot check box if you want to display the Phase Spectrum in the Wavelet Viewer.
  12. Select the Unwrap Phase check box if you want GeoCraft to unwrap the phase spectrum to a continuous phase.
  13. Type a value between 1 and 100 in the Amplitude Spectrum Threshold % field.
  14. Set the Minimum Phase value in degrees.
  15. Set the Maximum Phase value in degrees.
  16. Select the Add Regression Line check box to add regression lines to wavelet plots.
  17. Set the Minimum Regression Frequency.
  18. Click OK.
