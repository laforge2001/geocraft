Loading Data into GeoCraft

You must load data into GeoCraft before you can run algorithms or view data (such as volume or map plots). Use this procedure to load data into the GeoCraft Repository .

  1. Select File=>Load from the menu.
  2. In the Datastore Selection dialog box, expand the category of data (for example, 3D Seismic or Point Sets) that you want to load.
  3. Highlight the data type that you want to load and click OK.
  4. Navigate to the file system and directory where the data is stored.
  5. Highlight the data you want to load and click OK.

    6. GeoCraft will display a data properties dialog. If data properties (for example, data units) are missing, you will need to provide that information before you can load data into GeoCraft. If all data properties are known, you will need to verify the data properties before you load data into GeoCraft.

  6. Verify or set the data properties and click OK.

GeoCraft determines whether the data is in prestack or poststack format, and loads the data into the correct folder in the Repository . For example, the Grids folder contains horizons, and the 3D Seismic folder contains volumes.
