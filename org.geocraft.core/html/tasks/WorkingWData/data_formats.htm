GeoCraft Data Formats

GeoCraft can read, load, and write data in these formats to and from data stores. When GeoCraft loads data into the Repository , the program determines whether the data is in prestack or poststack format.

  • JavaSeis Volume
  • SEG-Y 3D Volume
  • CSV 3D Volume
  • ASCII Horizon
  • ModSpec Grid
  • ASCII PointSet
  • GOCAD PointSets
  • LAS Wells

GeoCraft is also able to display data in its viewers.
