Removing All Data from the GeoCraft Repository

GeoCraft provides an easy way to remove all data from the Repository in a single step; however, be sure that you want to perform this action because you cannot undo it.

Caution!: This action cannot be undone.

Click Reset to remove all data entities from the GeoCraft repository.
