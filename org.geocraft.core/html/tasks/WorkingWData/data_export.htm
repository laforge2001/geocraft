Exporting Data from GeoCraft

You can export data in the following data formats from GeoCraft and save the data to another location:

  • 2D survey lines
  • horizons (grids)
  • LAS well
  • OpenWorks well
  • volumes
  • wavelets

Exporting Data from the GeoCraft Repository

  1. Right-click the data entity in the repository view.
  2. Select Export.
  3. In the Datastore Selection dialog box, select the format for the data you are exporting and click OK.
  4. Navigate to the directory in which you want to save the data and click OK.
  5. Verify or set the properties for the data and click OK.

GeoCraft exports the data to the specified location.
