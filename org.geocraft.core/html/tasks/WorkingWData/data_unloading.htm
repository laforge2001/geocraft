Unloading a Single Data Entity from GeoCraft

In GeoCraft, the term unload means you are removing a data entity from the GeoCraft Repository. This data is not deleted from the data store from which it was originally loaded into GeoCraft.

Perform these steps to remove a single data entity from GeoCraft:

  1. In the Repository, select a data entity.
  2. Right-click the entity.
  3. Choose Unload from the popup menu.
