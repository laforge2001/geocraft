GeoCraft Troubleshooting

This topic contains brief descriptions of GeoCraft problems you may encounter, and ways to resolve those problems.

Description of Problem Explanation and Resolution
When I click Run in an algorithm panel, GeoCraft does not perform any action. The GeoCraft repository probably already contains a file with the name that would result from the output of the calculation (the value in the Output Name or Name field, in most algorithms). See GeoCraft File-naming Requirements for details.
I tried to edit a data set in GeoCraft, but could not.

Data loss or corruption can occur because:

  • the data set was edited in third-party software while GeoCraft was processing the data; see Editing Data Sets in Third-party Software While GeoCraft is Processing the Data for details or
  • you had two versions of GeoCraft running simultaneously, and you tried to edit or process the same data set in both running versions of GeoCraft at once. See Running Multiple Instances of GeoCraft at Once for details.
I want to view information in a window (such as Properties, the Repository, or the Error Log), but GeoCraft is not displaying that window. The window may have been closed by mistake. To resolve this issue, either reset GeoCraft to its default layout settings, or restore (open) the window that you want to use.
