Tip or Advice to Improve GeoCraft Performance Explanation or Further Details
Free unused memory to improve GeoCraft processing speed GeoCraft stores horizon data sets in memory as they are loaded. If your computer is running out of memory, unload data sets you are not using from the Repository.
