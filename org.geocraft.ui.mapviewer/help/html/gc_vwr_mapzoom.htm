Using the Zoom Features in the GeoCraft Map Plot

This topic describes how to use the zoom features in the GeoCraft Map Plot.

Before You Zoom in on a Plot

Perform the following steps, in order, before you use any of the zoom features:

  1. Load data into GeoCraft.
  2. Right-click a data entity in the Repository View.
  3. Select View => Map Plot.

Zooming in on a Plot

Click the Zoom in button

Zooming out on a Plot

Click the Zoom out button

Zooming in on an Area of a Plot

Hold down the SHIFT key, and zoom in on a specific region of the object in the plot window. GeoCraft draws a rectangle around that region, and zooms in on only that region within the plot window.

Returning an Objects in the Plot Window to their Original Size

When you are ready to return from zooming in on an area, click the Home View button on the toolbar. Home View returns the image to its original plot settings.
