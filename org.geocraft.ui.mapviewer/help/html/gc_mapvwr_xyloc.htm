Viewing the X,Y Location of the Cursor in the Map Plot

  1. View an object in the Map Plot.
  2. Ensure that the Layer model is shown toolbar button is selected, so that the Layers and Readout tabs display.
  3. Select the Readout tab.
  4. Move the mouse cursor across the object in the Map Plot.

Example: Readout Tab Displaying the Cursor Location in a Map Plot

The Readout tab displays the current x,y location of the mouse within the plot, and the row, column, and amplitude of the mapped object.
