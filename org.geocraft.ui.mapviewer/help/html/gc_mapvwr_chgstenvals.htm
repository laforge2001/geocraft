Changing the Start and End Values of an Object in the Map Plot

You may need to change the starting or ending values in the Map Plot to enhance the details of an object that you are viewing.

  1. View the object in the map plot.
  2. Move the mouse cursor over the object in the Map plot.
  3. Right-click the mouse.
  4. Click in the Start Value field, and type in a new value.
  5. Click in the End Value field, and type in a new value.
  6. Click OK.
