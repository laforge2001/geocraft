GeoCraft Map View

The GeoCraft Map View produces a 2D visual representation of a horizon in X, Y space. If more than one horizon is selected, GeoCraft displays all data in one plot with each grid stacked on top another. Map View also allows users to draw an area of interest (AOI) as a polygon as well as edit existing polygons. If seismic geometry is displayed in Map View, then inline, xline, or arbitrary lines of section can be selected and opened in Section View.

GeoCraft Map View Settings

By default, the GeoCraft Map View inherits settings from previous plots that you created of the same type. Subsequent plots that you create in the Map View will be displayed with the same settings, except for the Minimum and Maximum Color Scale Settings, within the Map View.

Tip: Add more horizons to the display by dragging-and-dropping the grids from the Repository into the map plot.
