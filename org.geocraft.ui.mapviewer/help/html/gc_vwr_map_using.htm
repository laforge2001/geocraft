Viewing Data in the GeoCraft Map Plot

Tip: Hover the cursor over any icon on the Map View toolbar to see tool tips for that icon.

  1. Load data into GeoCraft.
  2. Right-click a data entity in the Repository View.
  3. Select View => Map Plot.

GeoCraft launches a Map Plot viewer, and plots the data.
