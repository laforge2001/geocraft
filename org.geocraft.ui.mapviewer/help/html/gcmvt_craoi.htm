Creating and Editing an ASCII Area of Interest with the Map Plot View

Areas of Interest (AOIs) allow you to restrict processing within the defined area of a 2D or 3D data set. Use the Utilities menu to define 3D AOIs using an inline or xline range and increment, or 2D AOIs by using a shot point range and increment.

You can create an AOI in the Map View, and save it in an ASCII file format. If you create AOIs and save your session before you exit GeoCraft, your AOIs are restored when you return to that GeoCraft session.

Once the AOI is created and displayed in Map View, you may edit it by right-clicking on the AOI in the Layers tab and selecting Edit AOI Polygons. To move the polygon, hold the shift key down and left-click and hold one of the nodes, drag the polygon to the new position, and let go. To modify the polygon shape, hold the control key down and left-click on one of the nodes, dragging it to the new position. When done, exit the edit mode by clicking the middle mouse button. Then in the Repository right-click on the AOI and select Save to update.

Drawing an ASCII AOI with the Map View

Note: You must use a unique filename each time you create an ASCII AOI, or GeoCraft will display an error message, and the file will not be saved.

  1. View the data that you want to use to create an ASCII AOI in the Map View.
  2. Click the Define AOI icon (the first icon on the second row).
  3. Move the cursor onto the object in the Map Plot.
  4. Click MB1 to create an initial corner point for the AOI.
  5. Move the mouse and click MB1 to create additional corner points for the AOI.
  6. Optional. Hold down SHIFT and click MB2 to delete an existing point in the AOI.
  7. Click MB2 to complete (stop defining) the AOI.

    8. The Save ASCII AOI dialog box is displayed.

  8. Accept the default in the File Name field, or type in a new name.
  9. Click OK.

GeoCraft saves the AOI. You can find the AOI in the Areas of Interest folder in the Repository View, and in the Areas/Regions of Interest folder under the GeoCraft Viewer Layer Model and Readout Tabs.
