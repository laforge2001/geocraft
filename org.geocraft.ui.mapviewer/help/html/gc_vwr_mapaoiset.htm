Editing the Settings of an Area of Interest within the Map Plot View

You can create an area of interest (AOI) in the Map View, and save it in an ASCII file format. After you create the AOI, you can change its display settings.

  1. Create an AOI in the Map View. Or, you can load an existing AOI into GeoCraft.
  2. Right-click on the AOI name in the Layer Model tab, and select Settings from the popup menu.
  3. Type in a Transparency value. This value determines how opaque or transparent the AOI is within the Map View.
  4. Set the following Inclusive Fill Properties. These properties will be applied to everything that is set as inclusive within the AOI:
    • Fill Style: click the drop-down list and select Solid, Texture, or None
    • Fill Color: click in the box and select a color from the palette
  5. Set the following Inclusive Line Properties. These properties will be applied to everything that is set as inclusive within the AOI:
    • Line Style: click the drop-down list and select Solid, Dashed, or None
    • Line Width: type a value in the field
    • Line Color: click in the box and select a color from the palette
  6. Set the following Exclusive Fill Properties. These properties will be applied to everything that is set as exclusive within the AOI:
    • Fill Style: click the drop-down list and select Solid, Texture, or None
    • Fill Color: click in the box and select a color from the palette
  7. Set the following Exclusive Line Properties. These properties will be applied to everything that is set as exclusive within the AOI:
    • Line Style: click the drop-down list and select Solid, Dashed, or None
    • Line Width: type a value in the field
    • Line Color: click in the box and select a color from the palette
  8. Set the following Point Properties. These properties are applied to the end points of the AOI:
    • Point Style: click the drop-down list and select an option
    • Point Size: type a value in the field
    • Point Color: click in the box and select a color from the palette
  9. Click OK.

GeoCraft applies the new settings to the AOI, and closes the Settings dialog box.
