Viewing Multiple Wavelets in a Single Plot

You can view multiple wavelets in a single plot by selecting more than one wavelet for display. Each wavelet is assigned a different color. See the Hints section below to determine which color represents each wavelet. Frequency spectra parameters can be set up in Edit -> Preferences.

Selecting Multiple Wavelets for Viewing

  1. Access the GeoCraft Repository.
  2. Open the Wavelets folder.
  3. Hold down the SHIFT key and click the starting and ending files to select multiple wavelets that are grouped together in the Repository. Hold down the CTRL key and click individual files to select wavelets that are not next to each other.
  4. View the wavelets.

Hints and Tips About Using the GeoCraft Wavelet Viewer to View Multiple Wavelets

Use the following hints and tips to optimize your use of the viewer. This figure shows the Wavelet Viewer after selecting two wavelets. The Layer and Readout tabs are exposed on the phase spectra plot. This allows you to hide or show each wavelet and read the wavelet or spectra values. This feature can be used to determine which color corresponds to which wavelet.

Note: The phase spectra plot is optional; you are not required to display this plot in the Wavelet Viewer. See Setting Wavelet Plot Preferences for instructions about turning the display of the phase spectra plot on and off in the Wavelet Viewer.

It is helpful to know the following information when working with the Wavelet Viewer:

  • GeoCraft plots each wavelet with a different color
  • Toggle the Layer model is shown (hidden) button to show or hide the list of wavelets that are currently displayed
  • When the Layer model is shown button is selected, you can add or remove wavelets from the viewer by selecting their corresponding check box in the Layers tab
