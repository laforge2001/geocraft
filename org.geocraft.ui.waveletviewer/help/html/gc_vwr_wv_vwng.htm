GeoCraft Wavelet Viewer

GeoCraft provides a viewer that plots wavelets, and the amplitude and phase spectra of each wavelet. Multiple wavelets can be included in the same plot.

Note: The phase spectra plot is optional; you are not required to display this plot in the Wavelet Viewer. See Setting Wavelet Plot Preferences for instructions about turning off the phase spectra plot if you do not want to display it.

  1. Open the GeoCraft Repository.
  2. Expand the Wavelets folder.
  3. Right-click the wavelet you want to view.
  4. Select View => Wavelet Plot from the popup menu. GeoCraft launches the Wavelet Viewer.
  5. View the wavelet plot, amplitude and phase spectra (optional) of each wavelet that you selected for viewing.
