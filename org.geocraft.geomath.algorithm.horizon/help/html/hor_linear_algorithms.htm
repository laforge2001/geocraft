Linear Filter Algorithms

Linear filter algorithms remove noise from seismic data interpretations. This is also known as smoothing seismic interpretations.

Overview of Linear Filter Algorithms

Linear filters apply the same operations and weight to each input pixel of data within a seismic interpretation or a digital image; the filters do not adapt to the data. The filters work by convolution with a kernel (moving window) over the data set, so they are very fast. However, the filters are not very sensitive to the data, so everything is smoothed equally.

Linear Filter Algorithms Available in GeoCraft

GeoCraft provides the following linear filter algorithms:

  • Convolution (Linear Filters)
  • Kuwahara
  • Median Smooth
  • Symmetric Nearest Neighbor
  • Weighted Smooth
