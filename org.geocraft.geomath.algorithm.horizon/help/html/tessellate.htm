Interpolate Tessellate

The Interpolate Tessellate aglorithm fills in null values on a horizon.

Overview of the Interpolate Tessellate Algorithm

Interpolate Tessellate differs from other interpolation algorithms because it uses triangular tessellation to fill in null values within a horizon.

Using Interpolate Tessellate to Fill in Null Values in a Horizon

  1. Load horizon (grid) data into GeoCraft.
  2. Optional; Load Area of Interest data into GeoCraft.
  3. Select a Horizon.
  4. Optional; only perform this step if you want to limit the size of the horizon by using an area of interest in the calculation Select the Use Area of Interest? check box.
  5. Optional; Select an Area of Interest.
  6. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  7. Optional; Type additional information in the Comments field. Text from this field displays in the Properties view when an entity is selected in the repository.
  8. Click Run.

    9. GeoCraft runs the algorithm and stores the new horizon in the Repository, under the Grids folder.

  9. Use the GeoCraft Map Plot to view the results.
