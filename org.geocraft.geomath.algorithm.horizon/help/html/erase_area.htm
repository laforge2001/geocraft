Erase Area

You can use the Erase Area algorithm to create holes in a horizon.

Erase Area Overview

Use Erase Area to remove an Area Of Interest from a horizon.

Note: Nodes within the AOI are set to null. The size of the grid, or number of nodes, remains unchanged.

Erasing an Area of Interest from a Horizon

  1. Load horizon data into GeoCraft.
  2. Load Areas of Interest into GeoCraft, or create an Area of Interest for this calculation.
  3. Select a Horizon.
  4. Select an Area of Interest.
  5. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  6. Optional Type in Comments. This text will be displayed in the Properties view when the entity is selected.
  7. Click Run.
  8. Use the GeoCraft Map Plot to view the horizon with the area of interest erased from it.
