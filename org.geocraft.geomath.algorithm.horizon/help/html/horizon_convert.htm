Convert Horizon

Use the Convert Horizon algorithm to change the values of an existing horizon and create a new horizon.

Overview of the Convert Horizon Algorithm

The Convert Horizon algorithm allows you to select an existing horizon, and then change values associated with the horizon to create a new horizon.

Use these tips when you enter values for horizon parameters:

  • To create a finer view of the original horizon, increase the number of rows and columns in the output horizon.
  • To create a coarser view of the original horizon, decrease the number of rows and columns in the output horizon.
  • To change the rotation of the original horizon, enter a value in the Primary Angle field.

Note: The Convert Horizon algorithm only rotates the grid on which the horizon is displayed; it does not rotate the horizon. The horizon data itself does not move.

Converting a Horizon

  1. Load horizon data into GeoCraft.
  2. Select a Horizon. GeoCraft populates the values for each Input Horizon Parameters field.
  3. Type a new value in each Output Horizon Parameters field that you want to change.
  4. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  5. Optional Type Comments in the field provided. Comments display in the Properties view when the new horizon is selected in the Repository.
  6. Click Run.
  7. Use the GeoCraft Map Plot to view the results.
