GridLines

The Grid Lines algorithm extracts rows and columns (lines) from a horizon to make a sparse horizon. The output grid that the algorithm creates has the same number of nodes, and decimated grid nodes are set to null.

The Grid Lines and Decimate algorithms are similar, with the following exceptions:

  • Grid Lines uses an OR operation to pass node values; the Decimate algorithm uses an AND operation to pass node values
  • Grid Lines extracts lines, and Decimate extracts points
  • Grid Lines outputs row and column lines, and discards points

Overview of the GridLines Algorithm

GridLines generates a new grid from an existing horizon by selecting every nth row or column node. If you set the row spacing to 10 and the column spacing to 5, the algorithm will output every 10th row and every 5th column. All other node values will be set to null.

Note: The horizon may contain zeroes within the inline or xline; the algorithm is able to calculate Grid Lines for a new horizon with zeroes in either direction.

Running the Grid Lines Algorithm

  1. Load horizon data into GeoCraft.
  2. Select an Input Grid.
  3. Type in a Row Spacing value to specify the row increment.
  4. Type in a Column Spacing value to specify the column increment
  5. Type in the Output Grid Name. Make sure that the name complies with GeoCraft file-naming requirements.
  6. Optional Type in Output Comments. The comments are part of the metadata for the new horizon, and may be seen in the Properties view when the new horizon is selected in the Repository.
  7. Click Run.

    8. GeoCraft runs the algorithm, and the new horizon is populated into the Grids folder in the Repository.

  8. Use the GeoCraft Map Plot to view the results.

Note: The Grid Lines algorithm outputs row and column lines, and discards points.

Tip: Use the Table Viewer for quality control, to ensure that decimation was performed properly.

Example

In this example, set Row Spacing 10 and Column Spacingto 5.

If the current row number is evenly divisible by 10, then every node is passed through, independent of column number. (The entire row is output. If the current row is not divisible by 10, but the current column is divisible by 5, then the node value is passed through, independent of line number. (Every 5th column is output.)

Note: In the Grid Lines algorithm, if either Row Spacing or Column Spacing are set to 1, there will be no decimation. All node values will be output.
