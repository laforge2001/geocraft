Predefined Multiples Created by the Group Multiples Algorithm

The Group Multiples algorithm creates the predefined horizons listed here; the number of multiples that GeoCraft lists will depend upon the number of input horizons you selected during the initial calculation.

GeoCraft creates the multiples in Table 1 when 1 input horizons is selected for a group multiples calculation. Table 1 lists a value representing the horizon name and a value representing the corresponding equation.

Multiples Created when One Horizon is Input

Table 1: Multiples Created by GeoCraft from the First Input Horizon
