Linear Interpolation

The Linear Interpolation algorithm linearly interpolates interior nulls using neighboring values.

Linear Interpolation Overview

This algorithm searches for nulls within a horizon. If a null is found, Linear Interpolate searches up to the specified number of rows (xlines) and columns (inlines) for the nearest non-null value. The algorithm assigns weights to the distance in each direction from the point being interpolated. Nearer distances receive a higher weight, and farther distances receive a lower weight. The Linear Interpolation algorithm then averages the two weighted values to provide an output.

Generally, only interior nulls are interpolated. Linear Interpolate does not extend the horizon to the limits of its box (like the Extrapolate and Median Smooth algorithms do). However, if a portion of the horizon edge is concave, this area may be partially filled, depending on the value of the maximum row and column search range. If the search range is small, little will be filled.

Interpolating a Horizon

  1. Load horizon data into GeoCraft.
  2. Select a Horizon.
  3. Optional; only perform this step if you are using an Area of Interest in your calculation. Select the Use Area of Interest? check box.
  4. Optional Select an Area of Interest.
  5. Set the Maximum # of columns (inlines) to fill.

    6. Caution: This value defaults to zero, which uses the maximum number of columns possible (filling in all columns). If you set the value to 1, the algorithm does not interpolate data (infill) in any columns.

  6. Set the Maximum # of rows (xlines) to fill.

    7. Caution: This value defaults to zero, which means to use the maximum number of rows possible (filling in all rows). If you set the value to 1, the algorithm does not interpolate data (infill) in any rows.

  7. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  8. Optional Type additional text in the Comments field. This text is displayed in the Properties view for the entity.
  9. Click Run.
  10. Use the GeoCraft Map Plot to view the results.
