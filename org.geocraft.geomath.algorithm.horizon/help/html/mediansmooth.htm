Median Smooth

The Median Smooth algorithm uses a median filter to smooth a horizon. A median filter is a non-linear filtering technique that preserves the edges in maps while smoothing the data by removing highly varying values.

Overview of Median Smooth Algorithm

The Median Smooth algorithm uses a window specified by a number of inlines and a number of xlines. Values in the window are sorted into numerical order and the center value, or the median value, is selected for output.

Note: If you do not want to filter all data points in the horizon, you can use a mask horizon during the calculation. A mask is a horizon where processing is performed on the points that are not null. When Median Smooth finds nulls in the mask horizon, the algorithm copies values from the input horizon through to the output.

Smoothing a Horizon with the Median Smooth Algorithm

  1. Load horizon data into GeoCraft.
  2. Select a Horizon.
  3. Optional; only perform this step if you want to use a mask horizon in the smoothing calculation. Select the Don't Filter Entire Horizon? checkbox.
  4. Optional Select a Mask horizon.
  5. Select a Filter Option:
    • Filter based on columns/rows: creates a rectangle-shaped filter in the X and Y directions specified to the number of inlines and xlines you set. If you select this option, type values in the Filter Width in Columns and Filter Width in Rows fields.
    • Square filter based on number of points: creates a square filter (for example, 9 inlines by 9 xlines). If you select this option, type a value in the Filter Size field. When this length is reached, GeoCraft stops filtering and calculates the median value.
  6. Select an Interpolation Option:
    • Only values that are not null: uses only non-null values during interpolation and ignores null values
    • Only null values: uses only null values during interpolation and ignores non-null values.

      • Warning! This option will extend the edges of the horizon by half the filter length.

    • All values: uses null and non-null values during interpolation.

      • Warning! This option will extend the edges of the horizon by half the filter length.

  7. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  8. Optional Type text in the Comments field. This text is displayed in the Properties view for the entity.
  9. Click Run.
  10. Use the GeoCraft Map Plot to view the results.
