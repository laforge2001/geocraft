Horizon Merge

The Horizon Merge algorithm allows you to merge horizons by performing Minimum, Maximum, or OR operations on several horizons.

Note: You must use horizons with exactly the same geometry in your calculations to merge the horizons; if the geometries of the horizons differ, this algorithm will not work.

Overview of the Horizon Merge

Horizon Merge is similar to algorithms in the Basic Math (Minimum and Maximum) and Boolean Algebra (OR) algorithms, but you may select an unlimited number of horizons for this operation (the Basic Math and Boolean Algebra algorithms limit you to 2 input horizons).

Note: Once horizons are selected, the HorizonMinMaxOr allows you to change their order by moving them up or down in the list. Horizons are processed in a top-down order for the OR merge operation.

Horizon Merge does not modify the original input horizon(s).

How the Horizon Merge Algorithm Works

  1. For the OR operation the algorithm starts with the top-most horizon first and searches for the first non-null value in the list of input horizons at each node.
  2. For the MIN and MAX operations the algorithm scans all input horizons, ignoring order, and outputs either the minimum or the maximum value at each node.
  3. If all horizons are null, a null is output.

Performing a Horizon Minimum, Maximum, or OR Operation on Several Horizons in a Single Run

Caution: The horizons that you create during this procedure are held in memory only; if you do not save them before you quit GeoCraft, you will lose the data.

  1. Load horizon data into GeoCraft.
  2. Click the Add button next to the Horizons field.

    3. A Selection Dialog displays, listing all of the horizons that you have loaded into GeoCraft.

  3. Select the horizons you want to use in the calculation and click OK.
  4. In the Horizons field, select individual horizons and use the Up and Down buttons to move the horizons toward the top or bottom of the calculation order. GeoCraft gives preferential treatment to the top-most horizon in the list when trying to replace null values in other horizons within the list. If you selected a horizon you do not want, highlight the horizon and click Remove.
  5. In The Operation to Perform field, select Minimum, Maximum, or OR.
  6. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  7. Click Run.

    8. GeoCraft performs the operations on the selected horizons, and populates the new horizons into the Repository.

  8. Use the GeoCraft Map Plot to view the results.

You may now plot and view the data in a GeoCraft viewer.
