Extrapolate

The Extrapolate algorithm extends an existing horizon.

Overview of the Extrapolate Algorithm

The Extrapolate algorithm creates new data points in a horizon, and bases the new points on known, nearby points within the data set. Extrapolation is similar to interpolation (infill); however, extrapolation results are subject to greater uncertainty, and are often less meaningful.

Extrapolating a Horizon in GeoCraft

  1. Load horizon data into GeoCraft.
  2. Optional; only load this data if you will use an Area of Interest to limit the data set in which you are working. Load Area of Interest (AOI) data into GeoCraft.
  3. Optional; perform this step only if you want to use an AOI to limit the data set in which you are working. Select the Use Area of Interest? check box.
  4. Optional Select an Area of Interest.
  5. Select a Horizon.
  6. Accept the Parameters, or type in new values:
    • Top Distance - the number of rows to extrapolate
    • Bottom Distance - the number of rows to extrapolate
    • Left Distance - the number of columns to extrapolate
    • Right Distance - the number of columns to extrapolate
  7. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  8. Optional Type in Comments. The comments are part of the metadata for the new horizon, and may be seen in the Properties view when the new horizon is selected in the Repository.
  9. Click Run.

    10. GeoCraft runs the algorithm, and the new horizon is populated into the Grids folder in the Repository.

  10. Use the GeoCraft Map Plot to view the results.
