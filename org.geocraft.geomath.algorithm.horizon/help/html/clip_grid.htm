Clip Horizon

The Clip Horizon algorithm can be used to limit the data (ie. usually time or depth) range of a horizon.

Overview: Clip a Horizon Algorithm

When you use this algorithm, you create a new horizon by clipping values that are within the range of an existing horizon. Any values in the original horizon that are outside the range of values for the clipped horizon are changed to either nulls, the threshold values, or a user specified value.

This algorithm performs the clip function on 2D and 3D horizons.

The following rules apply with the Clip to Threshold option:

  • If a value is less than the minimum, the value is reset to the specified minimum threshold, to another value, or set to null.
  • If a value is greater than the maximum, the value is reset to the specified maximum threshold, to another value, or set to null.

If the Clip to a value option is selected, these rules also apply but values are reset to the user specified Set Value instead of the min or max threshold.

Clipping a Horizon

  1. Load 2D or 3D horizon data into GeoCraft.
  2. Select the Type of Horizon (2D or 3D) that you want to clip.
  3. Select a 2D/3D horizon.
  4. (Optional) Select the Use Area of Interest? check box.
  5. (Optional; only perform this step if you selected the check box in the prior step.) Select an Area of Interest.
  6. Type the minimum value to which the horizon should be clipped in the Clip Min field.
  7. Type the maximum value to which the horizon should be clipped in the Clip Max field.
  8. Select a Clip Type:
    • Clip to threshold forces out-of-range data to the Clip Min and Clip Max values
    • Clip to a value forces out-of-range (high and low) data to a value that you set
    • Replace with nulls applies null values to out-of-range data
  9. Optional; you will only perform this step if you selected the Clip to a value clip type. Type in the Set Value.
  10. Accept the Output Grid Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  11. Click Run.
  12. Use the GeoCraft Map Plot to view the results.
