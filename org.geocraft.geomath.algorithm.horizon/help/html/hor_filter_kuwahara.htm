Kuwahara Filter Algorithm

A Kuwahara filter reduces noise while preserving the edges of an image.

Overview of the Kuwahara Filter

Choose the Kuwahara filter when you need to isolate anomalies within a data set's attributes. Consider using a different filter if you have a data set with a lot of random noise; the Kuwahara filter does not handle random noise well.

The Kuwahara filter performs the following operations when it filters data:

  1. Divides the kernel support into four regions
  2. Selects the region with the lowest variance (the smallest spread of values)
  3. Returns the mean of the pixels within the selected region

This filter ensures that similar data contributes to the output, and that the edges are not averaged across the data set. The position and magnitude of edges is maintained, but smoother areas of the image tend to get blurred. The result is a flat image with sharp edges.

Tip: For the best results with a Kuwahara filter, use a mild median filter on your data set first, and then run the Kuwahara filter against the data set.

Smoothing Data with the Kuwahara Filter in GeoCraft

  1. Load horizon data into GeoCraft.
  2. Select a Horizon.
  3. Type a pixel filter value in the Size field.
  4. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  5. Optional Type additional text in the Comments field. Text typed in this field displays in the Properties view when the filtered horizon is selected in the Repository.
  6. Click Run.

    7. GeoCraft runs the algorithm and smooths the horizon. The resulting horizon is in the Repository, under the Grids folder, with its parent horizon.

  7. Use the GeoCraft Map Plot to view the results.
