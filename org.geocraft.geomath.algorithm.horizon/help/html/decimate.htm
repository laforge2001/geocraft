Decimate a Horizon

The Decimate algorithm extracts points in a grid to produce a sparse version of the original horizon. Decimated points in rows or columns are set to null. Decimate is similar to the Grid Lines algorithm except that it uses an AND operation instead of an OR operation to pass node values.

Overview of Decimate Horizon

Decimate generates a new grid from an existing horizon by selecting every nth row and every nth column node. If you set row spacing to 10 and column spacing to 5, the Decimate algorithm will output every 5th column value in every 10th row. All other node values will be set to null. An increment of 1 will output all points in that direction (that is, a row decimation of 1 and a column decimation of 5 will output all rows in every 5th column).

Note: The algorithm maintains the same number of nodes, and therefore the same number of rows and columns, as in the original horizon.

Example

For example purposes, set Row Spacing to 10 and Column Spacing to 5. If the current row number is evenly divisible by 10, then every column node that is divisible by 4 is passed through. All other node values are set null.

Decimating a Horizon

  1. Load horizon data into GeoCraft.
  2. Select a horizon to decimate in the Input Grid field.
  3. In the Row Decimation box, set a row increment.
  4. In the Column Decimation box, set a column increment.
  5. Accept the Output Grid Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  6. Optional Type text in the Output Comments field. This text will be displayed in the Properties view for the entity when the entity is selected.
  7. Click Run.
  8. Use the GeoCraft Map Plot to view the results.

Note: The Decimate algorithm discards row and column lines, and outputs row and column points, unless one increment is set to 1.

Tip: Use the Table Viewer for quality control, to ensure that decimation was performed properly.

Example of a Decimated Horizon

Use the following steps to produce an image similar to this one:

  1. Select a Horizon.
  2. Set the X and Y decimations to 5.
  3. Use the GeoCraft Map Plot to view the results.
