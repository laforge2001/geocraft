Weighted Smooth

Use this algorithm to smooth a horizon.

Overview of the Weighted Smoothing Algorithm

Typically, Weighted Smoothing is used to smooth interpolated horizons. Smoothing averages adjacent values in the horizon according to a mathematical formula, filtering out high frequencies in the data set. Nulls within the horizon are not smoothed. With the Weighted Smooth algorithm a second horizon may be used as a mask, so that GeoCraft smoothes only the masked area.

GeoCraft only applies the Edge Weight and Blending Horizon fields to the mask horizon if the mask horizon contains a depopulated grid. Use Edge Weight and Apply blending horizon fields as follows:

  • Edge Weight should be in the 0-1 range but it has a very minor effect
  • leave the Apply blending horizon check box blank to preserve grid lines in the mask horizon
  • select the Apply blending horizon check box to smooth the grid lines in the mask horizon

Smoothing a Horizon

  1. Load horizon data into GeoCraft.
  2. Select a Horizon.
  3. Optional; only perform this step if you are using a mask horizon in the calculation. Select the Don't Filter Entire Horizon? check box.
  4. Optional Select a Mask horizon.
  5. Optional Type in an Edge Weight value. The central value of the filter is 1, and the end points of the filter are the edge weight value. GeoCraft uses the Edge Weight as a taper so that the points further away from the point being smoothed are given less weight in the calculation.
  6. Optional Select the Apply blending horizon check box. Only select this check box if you applied a mask horizon in the calculation, and the mask has a depopulated grid.
  7. Type in a value for the Filter width in columns field.
  8. Type in a value for the Filter width in rows field.
  9. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  10. Optional Type in additional Comments. This text displays in the Properties view for the entity when the entity is selected in the repository.
  11. Click Run.
  12. Use the GeoCraft Map Plot to view the results.
