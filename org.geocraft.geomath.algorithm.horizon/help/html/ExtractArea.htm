Extract Area

Extract Area sets the horizon values that are outside of a specified Area of Interest to null.

Overview of Extract Area

When you use the Extract Area algorithm, GeoCraft sets horizon nodes that are outside the Area of Interest to null. Note that the grid size, or number of nodes in the horizon, remains the same.

Extracting an Area of Interest from a Horizon

  1. Load horizon data into GeoCraft.
  2. Load an Area of Interest (AOI), or create an Area of Interest from the horizon that you just loaded into GeoCraft.
  3. Select a Horizon.
  4. Select an Area of Interest.
  5. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.

    6. Caution: The value in the Output Name field must follow GeoCraft naming conventions.

  6. Optional Type text in the Comments field. Comments are displayed in the Properties view.
  7. Click Run.
  8. Use the GeoCraft Map Plot to view the results.
