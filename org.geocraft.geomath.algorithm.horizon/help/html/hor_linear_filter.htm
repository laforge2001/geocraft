Convolution (Linear Filters) Algorithm

The Linear Filters algorithm smooths horizons with the Gaussian, Square Mean, or Circular Mean filtering method.

Overview of the Linear Filters Algorithm

Linear filters give the same response in all parts of a data set; they do not adapt to the data. GeoCraft calculates linear filter size in pixels (the picture element). All GeoCraft calculation results from seismic horizon samples are measured in pixels.

Linear filters work by convolution within a moving window that is called a kernel. The input pixel is at the center of the kernel. The nonzero part of the kernel is the kernel support. Linear filters apply the same weights to all pixels in a data set, so they are very fast. However, linear filters smooth every pixel equally, so they do not perform sensitive smoothing on the data set.

Using the GeoCraft Linear Filter Algorithm to Smooth a Horizon

  1. Load seismic horizon data into GeoCraft.
  2. Select a Horizon.
  3. Select a Linear Filter:
    • Gaussian: applies weights according to the Gaussian distribution, where the kernel and degree of smoothing are controlled; however, noise is simply mixed into the result, and smoothing is random. For best results, remove spikes from a sample before you use a Gaussian filter to smooth the sample.
    • Square mean: a linear filter with a square-shaped support (the support is the nonzero part of the filter)
    • Circular mean: a linear filter with a circle-shaped support (the support is the nonzero part of the filter)
  4. Accept the sample Size, or type a new value in the field. Size is the number of inlines and xlines that are calculated for the sample.

    5. Tip: Size must be an odd number that is 3 or greater.

  5. Accept the Name, or type in a new one. Make sure that the name complies with GeoCraft file-naming requirements.
  6. Optional Type in Comments. Text typed in this field displays in the Propertyview when the new horizon is selected in the Repository.
  7. Click Run.

    8. GeoCraft runs the calculation, and creates the new, smoothed, horizon. The new horizon is in the Repository, under the Grids folder.

  8. Use the GeoCraft Map Plot to view the results.
