Generate Test Data

The Generate Test Data algorithm creates data sets for testing during the software development life cycle.

Use these instructions to generate fake data for software development testing:

  1. Move the spinner buttons up and down or type values in each field to specify the number of fake data sets you want to create for each category.
  2. Select or clear (check) the Show Picks? and Show Seismic Horizons check boxes.
  3. When you have specified parameters for the data you want to create, click Run.

A progress bar is displayed while GeoCraft creates the data sets. When the progress bar disappears, the data sets you requested are available in the Repository.
