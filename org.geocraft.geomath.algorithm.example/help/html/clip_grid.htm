Clipping a Horizon

Clipping produces a new horizon by isolating a portion of a horizon based on entered values. Where the values of a horizon are outside a user-specified range, those values are changed to the user-specified values. Values can be reset to minimum/maximum or to a specific value.

When minimum/maximum values are used, the following rule is applied:

  • If a value is less than the minimum, the value is reset to the specified minimum.

  • If a value is greater than the maximum, the value is reset to the specified maximum.

    Ensure you are in the GeoCraft perspective.

    Before you can use this algorithm, you must load horizon grid data into the Current Variables view.

    Choose Horizon from the Groups drop-down list.

    Select the Clip tab.

    Click the drop-down arrow next to the Horizon field, and select the horizon you want to clip.

    Choose a horizon from the Horizon drop-down list.

    Enter numeric values for the Clip Threshold Min and Max values.

    Select an option for out of range data:

      Clip to threshold forces out-of-range data to the minimum and maximum values

      Replace with nulls applies null values to out-of-range data

    Change the default name of the output variable if desired.

    Add a comment if desired.

    Click Run.

    Alternatively, you can use the interactive shell to clip to minimum/maximum values. See Writing Scripts.
