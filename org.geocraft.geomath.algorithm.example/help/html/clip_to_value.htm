Clip to a Value

When a specific clip value is used, the following rule is applied:

Where the values of a horizon are outside a user-specified range, sets the output to a user-specified value. That is, if any values are less than the minimum or more than the maximum, the values are reset to the specified value.

Ensure you are in the GeoCraft perspective.

To work with seismic data, one or more horizons must be loaded into the Current Variables view.

    Choose Horizon from the Groups drop-down list.

    Select downloaded variables from the drop-down list.

    Enter numeric values for Minimum and/ Maximum values.

    Enter numeric value for the Clip Value.

    Change the default name of the output variable if desired.

    Change the comment if desired.

Alternatively, you can use the interactive shell to clip to a specified value. See Writing Scripts.
