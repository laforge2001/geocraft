Volume Calculation

The Volume Calculation algorithm applies a simple linear transform to a single input volume over an area of interest (AOI).

Overview of the Volume Calculation Algorithm

Volume Calc applies a linear transform, Vout = (B1*Vin) + B2, where B1 and B2 are either constant values or horizons.

Using Volume Calculation to Perform Linear Transforms

  1. Load volume data into GeoCraft.
  2. Load Area of Interest data into GeoCraft.
  3. Optional Load horizon data into GeoCraft.
  4. Select a Volume.
  5. Optional; only perform this step if do not want to use an area of interest in the calculation. Clear the Use Area of Interest? check box.
  6. Select an Area of Interest.
  7. In the Parameters section, select an option and corresponding value for the B1 and B2 variables:
    • Use a constant; type a value in the Bn Value field
    • Use a horizon; select a Bn Horizon
  8. Accept the Volume Name, or type a new value in the field. If you leave the Volume Name field unchanged, GeoCraft appends vc to the input file name.
  9. Optional; Type additional information in the Comments field. Text from this field displays in the Properties view when an entity is selected in the repository.
  10. Click Run.

GeoCraft performs the calculation and writes the new volume to the data repository. You may view the data in the 3D viewer.
