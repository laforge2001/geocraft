MVXP Algorithm

The MVXP (Median Value trace eXPansion) algorithm performs median-based, windowed scaling on volumes.

CAUTION!: Some 2D data stores limit the number of characters in file names. For example, in some data stores the file name must be 9 characters or less. If you are working with 2D volume data from a data store with file name limitations, modify the GeoCraft Output Volume Name to comply with the data store requirements.

Overview of the MVXP Algorithm

MVXP computes the median trace absolute amplitude for a series of windows down the trace. A scale factor trace is derived by linearly interpolating these median values between the center of each window and then dividing the seismic trace by this scale trace. As a result each output trace will have an average median value near 1. Each trace is processed independently.

The original purpose of MVXP was to rescale seismic data so that floating point data volumes could be output as 8-bit volumes, conserving disk space. Using a scale factor, trace gain could be set so there was little or no clipping of high amplitudes while preserving low amplitudes. See Setting Scale Factor.

The following fields are optional, but you may find them useful:

  • Area of Interest (AOI) limits the amount of input data used in the calculation
  • Scale Factor takes all values in a trace and multiplies them by the value in the Scale Factor field
  • Clip Factor limits the amplitude range of input data used to compute the median

Using a Clip Factor

If you specify a Clip Factor in your calculation, values greater than the Clip Factor value are changed to match the clip factor value.

Tip: When the Clip Factor is set to 0, Clip is turned off and all trace values are included in the calculations.

Using MVXP to Scale Volumes

  1. Load volume data into GeoCraft.
  2. Optional Load an Area of Interest (AOI) into GeoCraft.
  3. Select an Input Volume.
  4. Optional; only perform this step if you will use an AOI during the calculation. Select the Use Area of Interest? check box.
  5. Optional Select an Area of Interest.
  6. Accept the Start Time, or type a new value in the field. This value specifies the start time of the trace in milliseconds.
  7. Accept the End Time, or type a new value in the field. This value specifies the end time of the trace in milliseconds.
  8. Accept the Scale Factor, or type a new value in the field.
  9. Optional; only perform this step if you will use a clip factor during the calculation. Select the Use a ClipFactor? check box.
  10. Optional Type a value in the Clip Factor field.
  11. Accept the Window Length (# of samples), or type a new value in the field.
  12. Accept the Output Volume Name, or type a new name in the field. If you are working with a 2D volume from a data store that has file-naming restrictions, modify the Output Volume Name to meet those restrictions.
  13. Click Run.

GeoCraft runs the calculation, and the new volume populates in the Repository under the 2D Seismic or 3D Seismic folder.

Use the Volume or Section Plot tools to compare the new volume with the original.
