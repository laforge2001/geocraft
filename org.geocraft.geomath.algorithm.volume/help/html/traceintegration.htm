Trace Integration

Trace Integration transforms existing reflection-seismic data into relative impedance.

Overview of the Trace Integration Algorithm

Performing trace integration on data can be helpful for observing cases like fining upward sands that do not have a top sand reflection. Trace integration with an operator length of one is simply a running sum of traces samples when the data are regularly sampled. However this process effectively boosts low frequency noise. As a result trace integration is often followed by a low cut bandpass filter. The Trace Integration algorithm in GeoCraft allows users to perform both processes in one step by setting the operator length larger than 1.

The Trace Integration algorithm works with 2D and 3D volume data.

Note: Trace integration should be applied to zero-phase broadband reflection data.

CAUTION!: Some 2D data stores limit the number of characters in file names. For example, in some data stores the file name must be 9 characters or less. If you are working with 2D volume data from a data store with file name limitations, modify the GeoCraft Volume Name to comply with the data store requirements.

Transforming Reflection Seismic Data into Relative Impedance

  1. Select the Type of Volume:
    • 2D: two-dimensional volume data
    • 3D: three-dimensional volume data
  2. Select a seismic Volume.
  3. Optional; only perform this step if you want to use an area of interest. Select the Use Area of Interest? check box.
  4. Optional; only perform this step if you want to limit the volume to a specific area of interest. Select an Area of Interest.
  5. In the Operator Length field, accept the value (80) in the field, or type in a new value.

    6. Operator Length defines the length of the integration operator.

    When Operator Length is set to 1 sample, the operator does a running sum of trace samples. This produces a 90-degree phase shift and a 1/f high cut. This effect is often seen as a ramp in time or a low frequency spike, which can be removed by applying a low-cut filter.

    When Operator Length is set to a value larger than 1, such as the default value of 80 samples, the operator is a sliding window that adds the nearest 80 trace samples. An Operator Length value of 80 greatly reduces the time ramp and removes low frequencies, so that you may not have to apply additional filtering to the calculation.

  6. Accept the value in the Output Volume Name field, or type in a new value.
  7. Click Run.

Use the volume plotting tools to view the results from the calculation.
